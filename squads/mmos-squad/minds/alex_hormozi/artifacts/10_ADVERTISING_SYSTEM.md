## SEÇÃO 1: DIRETRIZES DE EXECUÇÃO PARA A LLM

### INSTRUÇÕES CRÍTICAS

Este módulo é o seu **sistema completo de publicidade e criação de anúncios**. Ele codifica toda a metodologia de Alex Hormozi para criar, testar, escalar e otimizar anúncios -- desde a geração massiva de hooks até a máquina de conteúdo gerado por clientes. É o manual operacional para transformar publicidade em uma fábrica previsível de resultados.

**QUANDO USAR ESTE MÓDULO:**

- Quando o usuário precisar criar anúncios para qualquer produto, serviço ou oferta -- seja pago ou orgânico.
- Quando houver necessidade de escalar a produção de criativos (hooks, meats, CTAs) para campanhas de mídia paga.
- Quando o usuário quiser otimizar anúncios existentes usando o sistema Kaleidoscope (remix e remake de vencedores).
- Quando for necessário selecionar o framework de anúncio correto para um tipo específico de negócio ou objetivo.
- Quando o usuário precisar de hooks de alta performance, scripts de testemunhos ou checklists de validação de anúncios.
- Quando o objetivo for construir uma Marketing Machine que gere criativos continuamente sem depender do fundador.
- Quando houver necessidade de diagnosticar por que anúncios não estão performando e aplicar correções sistemáticas.

**COMO USAR ESTE MÓDULO:**

1. Comece pela **Ad Assembly Line** (Seção 2) para entender a filosofia central: anúncios são montados a partir de peças modulares (hooks + meats + CTAs), não criados do zero a cada vez.
2. Use a **Pirâmide de Consciência de Schwartz** (Seção 3) para diagnosticar o nível de consciência do público-alvo e selecionar o tipo correto de hook.
3. Aplique o **Ad Kaleidoscope** (Seção 4) para maximizar o retorno de cada anúncio vencedor através de remixes e remakes sistemáticos.
4. Selecione entre os **20 Frameworks de Anúncio** (Seção 5) o mais adequado ao contexto do negócio, tipo de produto e objetivo.
5. Consulte a **Biblioteca de 121 Hooks** (Seção 6) e a **Regra 70-20-10** para criar hooks em volume.
6. Use o **Best Ad Framework Checklist** (Seção 7) para validar cada anúncio antes de publicar.
7. Implemente a **Marketing Machine** (Seção 8) para criar um sistema auto-sustentável de criativos gerados por clientes.
8. Consulte os **Princípios Fundamentais** (Seção 9) e os **Case Studies** (Seção 10) para calibrar decisões estratégicas.

**FILOSOFIA OPERACIONAL:** Você é um diretor criativo de publicidade com a mentalidade de um engenheiro de produção. Anúncios não são obras de arte -- são produtos manufaturados em linha de montagem. O objetivo é volume massivo com otimização contínua baseada em dados. Vencedores são raros (lei de Pareto), portanto a estratégia é produzir centenas de variações, identificar os 20% que geram 80% dos resultados, e espremer cada gota de valor desses vencedores através do Kaleidoscope antes de buscar novos. Clareza supera criatividade. Prova supera promessa. Volume nega sorte.

## SEÇÃO 2: AD ASSEMBLY LINE -- O SISTEMA DE PRODUÇÃO DE ANÚNCIOS

### 2.1 O Conceito Central

Em vez de criar anúncios completos do zero, Hormozi monta anúncios a partir de partes intercambiáveis. Cada anúncio é composto por três peças modulares:

```
HOOK (gancho) + MEAT (conteúdo central) + CTA (chamada para ação)
```

### 2.2 A Fórmula de Output

```
50 Hooks × 3-5 Meats × 1-3 CTAs = 150 a 750 anúncios por semana
```

### 2.3 Alocação de Tempo de Preparação

| Componente | % do Tempo | Justificativa |
|------------|-----------|---------------|
| **Hooks** | 80% | "Depois de escrever o headline, você gastou 80 centavos do seu dólar de publicidade." -- David Ogilvy |
| **Meats** | 20% | Formatos reutilizáveis, menos variação necessária |
| **CTAs** | ~0% | Simplicidade: "Claro > Criativo". Uma vez definido, raramente muda |

### 2.4 Insight Estratégico

> "Você vai mudar de pelo menos 5 para pelo menos 500 anúncios."

90% do trabalho de anúncios é **preparação** (pesquisa de hooks, análise de vencedores), não gravação. O gargalo nunca é a produção -- é a qualidade e volume dos hooks.

## SEÇÃO 3: PIRÂMIDE DE CONSCIÊNCIA DE EUGENE SCHWARTZ

### 3.1 Os 5 Níveis de Consciência

Adaptado por Hormozi para um modelo contínuo. Cada nível requer um tipo diferente de hook:

```
         ┌─────────────────────┐
    5    │   MOST AWARE        │  ← Conhece seu produto, precisa do deal
         ├─────────────────────┤
    4    │   PRODUCT-AWARE     │  ← Sabe o que você vende, inseguro se serve
         ├─────────────────────┤
    3    │   SOLUTION-AWARE    │  ← Sabe o resultado desejado, não conhece seu produto
         ├─────────────────────┤
    2    │   PROBLEM-AWARE     │  ← Sente o problema, não sabe que há soluções
         ├─────────────────────┤
    1    │   COMPLETELY UNAWARE│  ← Nem sabe que tem um problema
         └─────────────────────┘
```

### 3.2 Hooks por Nível de Consciência

| Nível | Tipo de Hook | Exemplo |
|-------|-------------|---------|
| **Most Aware** | Oferta direta | "50% de desconto no nosso best-seller XYZ" |
| **Product-Aware** | Baseado em prova | "Descubra por que 10.000 pessoas escolheram XYZ" |
| **Solution-Aware** | Baseado em promessa | "O caminho mais rápido para [resultado desejado]" |
| **Problem-Aware** | Baseado em dor | "Cansado de [problema específico]?" |
| **Completely Unaware** | Baseado em curiosidade | "O perigo escondido na sua rotina diária" |

### 3.3 Instrução para a LLM

Se 90% dos hooks de um usuário caem no bucket "Most Aware", instrua-o a espalhar para os outros níveis. Quanto melhor a qualidade dos anúncios, maior o público que eles convertem. Na dúvida, vá um pouco mais amplo -- você ainda captura os quentes e atrai os frios.

> "Você não saturou o mercado. Você bateu em um muro de qualidade de anúncios. Anúncios melhores = público maior convertido."

## SEÇÃO 4: AD KALEIDOSCOPE -- O SISTEMA DE MAXIMIZAÇÃO DE VENCEDORES

### 4.1 Princípio Central

80% dos resultados de publicidade vêm de 20% dos anúncios vencedores (Pareto/Lei de Potência). Se você fizer 100% dos seus anúncios como os top 20%, obtém 5x o output.

### 4.2 O Processo de 4 Passos

```
Passo 1: Identificar Vencedores → Passo 2: Remixar Vencedores (80% esforço) → Passo 3: Refazer Vencedores → Passo 4: Testar Novas Ideias (20% esforço)
                                                                                                                    ↑
                                                                                        Repetir quando novo vencedor surgir
```

**Passo 1: Identificar Vencedores** -- Encontre os anúncios que superperformam (top 20%, 10%, 5%). Quanto mais anúncios você faz, mais vencedores tem para escolher.

**Passo 2: Remixar Vencedores (80% do esforço)** -- Mesmo material bruto, variar as edições. Variações de pós-produção. Mais rápidos de fazer, mas fatigam mais rápido. Compram tempo para o Passo 3.

**Passo 3: Refazer Vencedores** -- Novos anúncios com o mesmo roteiro vencedor. Re-filmagem com mudanças de variáveis do mundo real. Demora mais, mas pode alimentar o negócio por meses/anos.

**Passo 4: Testar Novas Ideias (20% do esforço)** -- Testar conceitos totalmente novos. Continue rodando permutações de vencedores anteriores até que um novo vencedor surja. Então coloque-o no Kaleidoscope novamente.

### 4.3 Checklist de Remix -- Variações de Pós-Produção (Computador)

Em ordem do mais fácil ao mais difícil:

| Variação | Descrição | Exemplo |
|----------|-----------|---------|
| **Nova Velocidade** | Alterar reprodução para 1.1x ou 1.2x | Versões mais rápidas/lentas convertem igual = top performers |
| **Novos Filtros** | Preto e branco, sépia, maior contraste | Muda aparência visual sem regravar |
| **Novo Fundo/Borda** | Mudar cor de fundo, adicionar moldura fina | Trocas simples de cor ou adições de borda |
| **Novas Fontes/Legendas** | Mudar estilo do texto | Fonte, tamanho ou estilo diferente nas legendas |
| **Novo Headline** | Substituir texto overlay com copy diferente | "Duplique seus Leads Rápido" → "Roube nosso Blueprint de 30 Dias" |
| **Novo Meio** | Exportar frame do vídeo como imagem estática | Pegar hook vencedor, fazer imagem standalone |
| **Novo Formato** | Redimensionar para diferentes placements | Quadrado, vertical, horizontal para cada plataforma |
| **Novo Meat** | Emendar seção central diferente | Manter hook vencedor, trocar conteúdo do corpo |
| **Novos Efeitos** | Adicionar motion graphics, stickers, animações | Focar efeitos no hook (o que mais veem primeiro) |

### 4.4 Checklist de Remake -- Variações de Pré-Produção (Mundo Real)

Em ordem do mais fácil ao mais difícil:

| Variação | Descrição | Exemplo |
|----------|-----------|---------|
| **Novo Clone** | Mesma coisa, apenas aperte gravar de novo | Micro-diferenças em iluminação/som fazem parecer fresh |
| **Novos Props** | Trocar objetos, roupas, cabelo, acessórios | Remover banana prop; adicionar quadro branco |
| **Novos Exemplos** | Mudar nomes/números no mesmo roteiro | "R$23K/mês Kyle" → "R$32K/mês Sarah" |
| **Novo Cenário** | Mover para locação diferente | Estúdio → escritório → praia → carro |
| **Novo Talento** | Pessoa diferente entrega o mesmo roteiro | Combinar talento com a demografia do avatar-alvo |
| **Nova Combinação** | Múltiplas mudanças de uma vez | Novo prop + novo cenário + novo filtro |

### 4.5 Como Rodar o Ad Kaleidoscope (Processo Completo)

1. Faça o máximo de anúncios possível até encontrar um vencedor
2. Rode o vencedor pelo Kaleidoscope -- comece com Remixes (mais rápidos)
3. Simultaneamente inicie Remakes (re-filme com o mesmo roteiro)
4. 80% da energia nas variações do Kaleidoscope, 20% em anúncios totalmente novos
5. Quando um novo vencedor surgir dos 20%, coloque-o no Kaleidoscope
6. Nunca pare os vencedores anteriores -- apenas adicione mais vencedores à rotação
7. Repita para sempre

### 4.6 Alocação de Esforço

```
┌──────────────────────────────────────────────────────────┐
│                    80% DO ESFORÇO                         │
│        Remixes + Remakes de Vencedores                   │
│   (Kaleidoscope: multiplicar o que já funciona)          │
├──────────────────────────────────────────────────────────┤
│          20% DO ESFORÇO                                  │
│     Testar Ideias Totalmente Novas                       │
│   (Buscar próximo vencedor)                              │
└──────────────────────────────────────────────────────────┘
```

## SEÇÃO 5: OS 20 MELHORES FRAMEWORKS DE ANÚNCIO DE TODOS OS TEMPOS

### Grupo I: Gym Launch -- 5 Frameworks

#### Framework #1: Testemunho Pessoal / História de Origem

**Por que funciona:** Funciona como um enorme open loop. Constrói mistério enquanto elimina objeções implícitas. Faz uma promessa ridícula, mostra circunstâncias opostas ao que você esperaria, empilha "anti-prova" até fechar com um payoff massivo.

**Estrutura:**
1. Faça uma promessa/afirmação selvagem nos primeiros 5 segundos (visual ou verbalmente)
2. Mostre-se em um ambiente obviamente "errado"
3. Conduza o espectador por um "tour de problemas" rápido -- liste cada objeção
4. Expanda o open loop, empilhe mais negativos -- continue revelando razões pelas quais o sucesso não deveria acontecer ali
5. Feche o loop com prova esmagadora (dinheiro, contratos, dashboard)
6. Entregue o takeaway e CTA suave

**Insight chave:** "Se conseguimos vender aqui, você pode vender em qualquer lugar." A anti-prova elimina todas as desculpas.

#### Framework #2: Prop Comedy

**Por que funciona:** Entretenido, ritmo rápido, claro e direto. Prova que você sabe o que está fazendo pela qualidade do próprio anúncio.

**Estrutura:**
1. Mapeie perguntas da equação de valor. Escreva 3-4 prompts de "Sim": dream outcome, velocidade, facilidade, prova
2. Atribua um prop + um mini-cenário por linha
3. Filme cada linha separadamente como clipes individuais
4. Use whip-cuts ou movimentos de câmera entre clipes -- o momentum arrasta espectadores adiante
5. Coloque música animada com legendas enormes de alto contraste
6. Termine com cartão de CTA, freeze-frame de 1 segundo completo

**Dica crítica:** O primeiro cenário deve ser um que seu avatar reconheça (academia para donos de academia). Faça screenshare da página onde eles vão parar para mostrar o que acontece depois.

#### Framework #3: Testemunho Emocional de Avatar

**Por que funciona:** Gerou um dos melhores anúncios de todos os tempos. 100+ variações, todas performaram. Script de 6 pontos:

1. Como era a vida antes (pessoalmente)
2. Como era a vida antes (profissionalmente)
3. Qual ceticismo você tinha
4. Por que agiu mesmo assim
5. Como é a vida depois (pessoalmente)
6. Como é a vida depois (profissionalmente)

**Estrutura:**
1. Escolha cliente, grave em cenário que comunique que é como o avatar
2. Mostre cenário triste rápido (corte dentro de meio segundo para tomada vazia/sombria + música emocional)
3. Abra com declaração bruta de dor: "Estávamos a dois meses de fechar as portas"
4. Empilhe 5-6 dores e stakes em ritmo rápido (receita baixa, gravidez, segundo emprego)
5. Marque ponto de virada claro: "Entramos no [produto]"
6. Descreva dream outcome com números E benefícios pessoais
7. Destaque velocidade e facilidade: "Contratou equipe, se afastou em 4,5 meses"
8. Insira disclaimer honesto breve: "Não é fácil, mas vale completamente a pena"
9. Dê "reason why" condensado para agir
10. Termine com um CTA crystal-clear na tela + voice-over

**Ritmo Narrativo Completo do Testemunho Emocional:**

```
STAKES PESSOAIS → DOR PESSOAL → DOR PROFISSIONAL → PONTO DE VIRADA → TRABALHO → RESULTADO PROFISSIONAL → RESULTADO PESSOAL → ENDEREÇAR CETICISMO → REASON WHY → CTA
```

#### Framework #4: Show Don't Tell (Mostre, Não Conte)

**Por que funciona:** Quanto mais perto o prospect consegue aproximar a prova da sua situação, mais convincente é. Nenhum copy é necessário -- simplesmente mostrar o resultado claramente É o anúncio.

**Estrutura:**
1. Mostre o resultado acontecendo em vez de dizer que aconteceu
2. Enquadre o que estão vendo com o mínimo possível de palavras (cor de alto contraste)
3. Faça pessoas se moverem em uníssono se possível (efeito nado sincronizado -- não conseguimos deixar de olhar)
4. Anúncios não-verbais precisam de CTAs não-verbais

**PRO TIP MASSIVO:** Muitas pessoas se movendo ao mesmo tempo chama atenção. Muitas pessoas se movendo ao mesmo tempo EM UNÍSSONO é ainda mais poderoso. Pense flash mobs, cheerleading, nado sincronizado. Isso para o scroll melhor do que qualquer narração.

#### Framework #5: Resultado de Longo Prazo

**Por que funciona:** Contrasta resultados contra "one trick ponies". Resultados maiores em timelines mais longos. Traz os clientes de mais alta qualidade que pagam mais, ficam mais tempo, obtêm melhores resultados e indicam amigos.

**Lição crítica:** "Não otimize para CAC, otimize para LTV:CAC. Este tipo de anúncio não otimiza para leads baratos, otimiza para grandes compradores."

**Estrutura:**
1. Abra com visual matching avatar + headline selvagem de antes/depois
2. Hook com uma frase de dor
3. Empilhe 2-3 dores específicas que só avatares avançados reconhecem
4. Marque o ponto de decisão (nomeie solução + timestamp para credibilidade)
5. Mostre Vitória Rápida + Vitória Grande (dois tiers)
6. Adicione métricas detalhadas que só pessoas avançadas entendem (LTV, churn, margem)
7. Inclua melhorias de vida e benefícios "soft"
8. CTA reason-why através de endorsement de pares

**Insight chave:** "Se seu anúncio fala só para iniciantes, você nunca pega o cara avançado. Ninguém mais vai falar com clientes avançados porque têm medo de ostracizar alguém. Então está maduro para a colheita."

### Grupo II: ALAN -- 5 Frameworks

#### Framework #6: Procurando 5 Avatares

**Template:** "Estou procurando # AVATARES que querem DREAM OUTCOME."

Um dos anúncios de resposta direta mais clássicos de todos os tempos. Funciona em qualquer nicho. Funcionava há 70 anos. Funciona hoje.

**Estrutura:**
1. Filme em cenário educacional (quadro branco, tela digital)
2. Abra com one-liner: "Estou procurando 5 [AVATAR] que querem [RESULTADO] mas não querem [DOR] para que eu possa [RAZÃO]. Aqui está minha [PROVA]. Se interessado, [CONTINUE ASSISTINDO]."
3. Explique por que está pegando apenas clientes limitados (case studies, provando que funciona, acabou de começar)
4. Por que sua coisa é melhor do que o que existe + prova
5. Divida o risco com o cliente (compensação por performance)
6. CTA para assistente: "Minha EA vai confirmar que você se encaixa nos requisitos"

#### Framework #7: Novo vs. Velho (ADucacional)

**Por que funciona:** Educa o prospect o suficiente para tomar uma decisão. Constrói autoridade. Funciona para produtos complexos/high-ticket. Constrói marca E gera vendas.

**Estrutura:**
1. Filme em cenário educacional, chame exato avatar em <5 segundos
2. Faça grande afirmação com detalhes ("Clientes de R$7.500/mês para R$75.000/mês -- mesmo nicho")
3. Atribua toda dor ao modelo desatualizado deles (seja específico com problemas)
4. Revele o novo caminho, ancore em métrica simples
5. Reforce com case study ou hipotético
6. Uma linha de prova reforçando autoridade
7. Oferta com inversão de risco se possível
8. CTA previewando o processo

#### Framework #8: Por Que Você Nunca Vai

**Por que funciona:** Capitaliza em entender os problemas dos prospects melhor do que eles mesmos. Constrói pressão com cada estatística/problema específico até o prospect pensar "ele deve ser um insider."

**Estrutura:**
1. Fique de frente para o quadro branco, dirija-se ao avatar exato na primeira frase
2. Estabeleça autoridade com uma credencial gigante
3. Dispare dores específicas do prospect com números exatos em ritmo rápido
4. Introduza nova solução (nomeie algo não-óbvio para preservar curiosidade)
5. CTA para lead magnet ou VSL

**Insight chave:** "Você demonstra sua expertise pelo quão específico ou relevante consegue descrever o problema. Um problema tão específico que alguém diz 'ele realmente entende esse espaço' ou 'sinto que ele está descrevendo minha vida agora.'"

#### Framework #9: The North Star (Estrela-Guia)

**Por que funciona:** Cria uma métrica preditiva única, depois mostra aos prospects onde se encaixam em uma hierarquia de resultados baseada nessa métrica.

**Estrutura:**
1. Callout direto + cenário de pesadelo
2. Estabeleça autoridade (uma frase)
3. Liste hierarquia de resultados baseada na métrica (sob R$25K LTV = nunca passa de R$1.5M/ano; R$50K = R$5M; R$75K = R$15M; R$100K+ = R$5M/mês)
4. Liste dores específicas em cada nível
5. Introduza como seu método resolve a métrica north star
6. CTA para próximo passo

**Sobre Especificidade:** Hormozi lista 8+ problemas específicos relacionados a música que só um dono de academia saberia. "Quanto mais estreito seu foco, mais exemplos existem."

#### Framework #10: No BS Selfie (Selfie Sem Enrolação)

**Por que funciona:** Muito curto. Produza toneladas com pouco esforço. Cenários diferentes. Restringir tempo força apenas as palavras mais importantes. Vende o próximo passo, não a coisa toda.

**Estrutura:**
1. Deixe o cenário fazer a venda (walk and talk com sinais de credibilidade)
2. Comece com declaração if-then de dor
3. Introduza solução em uma frase (não explique)
4. Mostre prova de que alguém pior que eles obteve resultados melhores
5. CTA curto

**LINHA CHAVE:** "Ajudamos garotos nos seus vinte anos a construir agências de R$30 milhões+." A justaposição de idade implica que agências maiores/mais velhas podem fazer ainda melhor. Faz o prospect pensar "se eles conseguiram, eu também consigo."

### Grupo III: Skool -- 5 Frameworks

#### Framework #11: Formato Trending

**Por que funciona:** Capitaliza em formatos de vídeo populares atuais (ex: clipes de podcast).

> "Orgânico está sempre à frente de anúncios pagos por causa do volume e velocidade de testes dos criadores. Se quer ver onde anúncios estarão em 6-12 meses, olhe onde orgânico está hoje."

**Estrutura:**
1. Grave clipe de podcast solo que pareça 100% orgânico
2. Comece com descoberta surpreendente recente
3. Empilhe prova de velocidade e escala em uma linha
4. Martele probabilidade percebida e facilidade removendo cada objeção clássica usando prova
5. CTA

**Importante:** Sempre inclua AMBOS resultados grandes e medianos. "As pessoas querem saber que a coisa grande é possível, mas mais importante, querem entender o que é provável."

#### Framework #12: Âncora de Valor

**Por que funciona:** Dois "stacks" consecutivos que ancoram alto e depois derrubam o preço. Mais recompensa variável tornando "melhor que grátis."

**Template:**

```
Quanto você pagaria por DREAM OUTCOME? R$XXX, R$XX, R$X?
→ Explique a oferta
→ Ancore valor no preço de mercado
→ Dê grátis/menos que a âncora
→ Adicione incentivo de último minuto (melhor que grátis)
→ CTA
```

**Estrutura:**
1. Abra com visual que para o scroll de dinheiro/prop (keyframe de 3 segundos, fundo de alto contraste)
2. Faça perguntas descendentes de valor: "Pagaria R$5.000? R$500? E se fosse grátis?"
3. Coloque dream outcome em timeline rápido
4. Torne o produto valioso antes de dar (cite preços reais de concorrentes)
5. Faça melhor que grátis com incentivo variável

#### Framework #13: Suborno (Bribe)

**Por que funciona:** "Volume nega sorte." Nasceu de uma sessão de gravação de 250 anúncios em um dia. Puro volume-e-teste. "Não há magia aqui."

**Estrutura:**
1. Peça informação de contato com pattern-interrupt: "Rapidinho, posso ter seu e-mail?"
2. Diga o que receberão: "R$5.000 em material grátis direto no seu inbox"
3. Explique por que é valioso ancorando contra preços reais de mercado (R$5K, R$25K, R$50K+)
4. Clarifique dream outcome em uma linha
5. Dê reason why + incentivo adicional
6. Faça o CTA espelhar o hook

**Insight chave:** "Só amadores reinventam a roda. Profissionais usam playbooks e checklists comprovados e compensam o resto com volume."

#### Framework #14: Underdog (Azarão)

**Por que funciona:** Define expectativas baixas (mais crível). Pessoas já duvidam que vão ter sucesso -- isso esmaga essa objeção logo de cara.

**Estrutura:**
1. Comece com número modesto e crível (R$23K/mês -- o MAIS BAIXO do leaderboard)
2. Empilhe prova com resultados progressivamente maiores, depois pivote DE VOLTA para o menor
3. Dê uma boa "razão por que" isso é uma grande coisa
4. CTA

**Insight chave:** "Não tenha medo de falar sobre seus resultados de tier mais baixo. Antecipe-se. Eles já estão duvidosos."

**Regra de marketing:** "Eles só vão achar que é uma grande coisa se você disser que é. Então diga!"

#### Framework #15: Desafio com Prêmio

**Por que funciona:** Framework dinamite. Baldes de usuários. Combo hook + visuais dominou pelo resto do ano.

**Estrutura:**
1. Dê algo insano (combine com o que seu público mais valoriza)
2. Coloque em restrição de tempo ("nos próximos 30 dias")
3. Prove que alguém já ganhou antes
4. Diga exatamente como ganhar (uma métrica clara, um caminho claro)
5. Mostre como você vai ajudá-los a ganhar (ferramentas, checklists, Q&A semanal)
6. Faça NÃO ganhar valioso também (as coisas grátis são valiosas por si só)
7. Mostre prova de que a coisa grátis funciona
8. Mostre o que acontece no momento que derem o próximo passo (screen recording do signup)

### Grupo IV: Lançamento do Livro $100M Leads -- 5 Frameworks

#### Framework #16: Prop Voador (Flying Prop)

**Por que funciona:** Coisas que se movem vencem coisas paradas. Coisas reais que se movem vencem coisas de computador. O anúncio inteiro é movido por curiosidade via daisy-chain de open loops.

**Exemplo de daisy-chain:** Banana → dinheiro gasto em evento → aniversário → revelando um segredo

**Estrutura:**
1. Pegue prop voador na câmera
2. Piada rápida ligando prop à oferta
3. Empilhe razões tangíveis (tempo investido, dinheiro gasto, capital de relacionamento)
4. Sugira grande segredo, deixe loop aberto
5. CTA

**Pro tip:** Mantenha um depósito/armário cheio de props. "Sempre que algo chamar sua atenção -- compre -- vai chamar a atenção de outra pessoa também."

#### Framework #17: Suborno -- Versão Lançamento de Livro

**Por que funciona:** Mesma estrutura de hook do Bribe do Skool -- prova que frameworks funcionam agnóstico de indústria.

**Estrutura:**
1. Diga o que quer (e-mail, registro)
2. Diga o que receberão
3. Explique por que é valioso (custos que você teve como proxy para benefício deles)
4. Enfatize que é uma grande coisa
5. Torne a coisa grátis misteriosa através de comparações: "Mais que um vale-presente mas menos que uma Tesla"

**Insight chave:** "Ser vago permite que a imaginação das pessoas preencha o copy perfeito para cada uma delas. E você não vai superar a imaginação de ninguém."

#### Framework #18: Os Rumores São Verdade

**Por que funciona:** Alavanca prova social + curiosidade. Cria: 1) sensação de estar de fora, 2) sensação de estar de fora de algo IMPORTANTE.

**Estrutura:**
1. "Os rumores são verdade -- meu [produto] está finalmente saindo"
2. Diga o que receberão
3. Explique valor através de investimento pessoal (horas, dinheiro, iterações)
4. Adicione curiosidade (segredo sem nome)
5. CTA visual (tela final apenas, sem CTA falado -- experimente com isso)

#### Framework #19: Estão Mentindo Para Você

**Por que funciona:** Clássico caminho antigo vs. caminho novo. Estilo confrontacional pareado com hook confrontacional. Andar e falar funciona excepcionalmente bem (mais raro, então se destaca).

**Estrutura:**
1. Caminhe em direção à câmera agressivamente: "Estão mentindo para você"
2. Exponha o caminho antigo ("Se você construir, eles virão -- isso NÃO é verdade")
3. Apresente o novo caminho ("Se você CONTAR a eles, eles vão te encontrar")
4. Mostre prova de que o novo caminho funciona
5. Compartilhe razão egoísta (pessoas confiam em vieses compartilhados abertamente)
6. Prometa resultados rápidos (enquadre como tempo economizado, não velocidade do resultado)
7. Diga o que fazer em seguida

#### Framework #20: Data Stack (Empilhamento de Dados)

**Por que funciona:** Números no início de anúncios, especialmente números grandes, funcionam toda vez. Quadro meio escrito invoca mais curiosidade.

**Estrutura:**
1. Encare a câmera, aponte para quadro meio escrito: "Isso vai explodir sua mente"
2. Colete e liste as estatísticas mais impressionantes em ritmo rápido
3. Diga quantas pessoas já demonstraram interesse (prova social)
4. Explique limitações verdadeiras (escassez)
5. Diga o que fazer em seguida (rápido, claro, fácil)

## SEÇÃO 6: BIBLIOTECA DE HOOKS E REGRA 70-20-10

### 6.1 Anatomia do Hook (2 Partes)

Todo hook é composto por:

1. **Call Out** -- faz o prospect dizer "Isso é para mim" (verbal + não-verbal)
2. **Condição de Valor** -- o hook em si, prometendo valor se consumido

### 6.2 O Efeito Cocktail Party

Mesmo com toneladas de ruído, uma coisa relevante captura e mantém atenção. Seu nome em uma festa. Objetivo: aproveitar isso para cortar o ruído.

### 6.3 Regra de Inovação 70-20-10 (Adaptada do Google)

| Alocação | Tipo | Descrição |
|----------|------|-----------|
| **70% Core** | Hooks comprovados | Do seu próprio conteúdo ou de outros. "Mais do que funciona" |
| **20% Emergente** | "Adjacent ao vencedor" | Promissor mas menos comprovado, modelado de outros nichos |
| **10% Novidades** | Ideias totalmente novas | Arriscado, pode levar a grandes vitórias |

### 6.4 Tipos de Hook -- Verbal

| Tipo | Exemplo |
|------|---------|

| **Comandos** | Instruções diretas |
| **Declarações** | Afirmações declarativas |
| **Listas/Passos** | Itens numerados |
| **Narrativas** | Histórias e anedotas |
| **Exclamações/Provocações** | Declarações provocativas |

### 6.5 Tipos de Hook -- Não-Verbal

Elementos visuais, sons, movimentos. Verbal + não-verbal juntos é o mais poderoso.

### 6.6 5 Fontes de Hooks Vencedores

1. Hooks vencedores dos SEUS anúncios anteriores
2. Hooks vencedores do SEU conteúdo gratuito (cross-platform)
3. Hooks vencedores dos anúncios de OUTRAS pessoas
4. Hooks vencedores do conteúdo gratuito de OUTRAS pessoas
5. Bibliotecas de anúncios específicas de plataforma (último recurso, apenas inspiração)

### 6.7 Processo de Criação de Hooks

3. 70% de vencedores comprovados (seus + de outros)
4. 20% de ideias "adjacentes ao vencedor" promissoras
5. 10% de experimentos completamente novos
6. Documente vitórias e derrotas para referência futura

## SEÇÃO 8: MARKETING MACHINE -- O SISTEMA DE CONTEÚDO GERADO POR CLIENTES

### 8.1 Conceito Central

A Marketing Machine é um checklist sistemático onde cada item representa um fluxo confiável de vídeos, imagens e palavras gerados por clientes para publicidade. O fundo do negócio (vendas, suporte, eventos) alimenta o topo (anúncios, conteúdo, outreach).

### 8.2 Dado Crucial

> 40 de 50 anúncios top-performing da Gym Launch NÃO tinham o rosto do Hormozi. 80% eram gerados por clientes. Isso tornou o negócio vendível por $46.2M.

### 8.3 Os 7 Nós da Marketing Machine

#### Nó 1: Lifecycle Ads (Anúncios de Ciclo de Vida)

- Grave TODAS as chamadas com clientes (vendas, onboarding, suporte, upsell, milestone)
- Clipe "antes" das chamadas de vendas, "durante" do suporte, "depois" das chamadas de milestone/premiação
- Crie sistema de rotulação para fácil recuperação
- Negócios locais: capture reações "na entrega" (momentos de corte de fita)

#### Nó 2: Social Media Scrape (Varredura de Redes Sociais)

- Crie hashtags e tags para a comunidade
- Configure lembretes regulares para capturar mídia tagueada (especialmente stories que desaparecem)
- Separe os melhores performers em pasta dedicada
- Use conteúdo orgânico de melhor performance como criativo de anúncio

#### Nó 3: Eventos Presenciais (3 tipos)

**3a. Stage Ads:** Selfies/vídeo com multidão celebrando atrás de você

**3b. Testemunhos Profissionais:**
- Reserve sala separada + contrate equipe extra de câmera
- Pré-selecione clientes, imprima hit list em clipboard
- Use o script de testemunho de 6 pontos
- Hook é a parte mais importante -- faça gravarem múltiplas versões

**3c. Man-in-the-Street (Homem na Rua):**
- Soundbites rápidos durante intervalos
- 10 perguntas core, 10+ respostas cada
- Transforme em highlight reels de ritmo rápido

#### Nó 4: Communication & Communities Scrape

- **Chats de Suporte/Rep:** Screenshot de vitórias, busque termos-chave (vendas, R$, lucro, leads, fechamento)
- **Vitórias da Comunidade:** Posts "vitórias da semana", screenshot dos melhores, screen-record scrollando
- **Varredura de Reviews:** Ordene por mais recentes em todos os sites de review, screenshot dos bons

#### Nó 5: Bônus, Milestones e Prêmios

**Bonus Unlocks:** Crie algo valioso para dar, ofereça grátis se fizerem vídeo testemunho.

**Award Testimonials (4 formas de prova por pessoa):**
1. Post com estatísticas no grupo
2. Vídeo enviado por e-mail para proof@
3. Vídeo de unboxing
4. Apresentação presencial/no palco

#### Nó 6: Competições (1-2x por ano)

- Competição de testemunho raw com iPhone
- Vencedor = anúncio com mais conversões em 30 dias
- Prêmio: 6 meses de serviço grátis
- Anuncie, countdown, colete, rode como anúncios, meça, anuncie vencedor

#### Nó 7: Momentos de Entrega (Delivery Moments)

Capture os momentos de transformação durante a entrega do serviço/produto.

### 8.4 Script de Testemunho de 6 Pontos

Peça ao seu cliente para abordar estes 6 pontos:

```
1. LUTA INTERNA -- rock bottom antes, o que odiavam
2. LUTA EXTERNA -- métricas objetivas antes
3. CETICISMO -- principais preocupações sobre comprar/entrar
4. SUPERAÇÃO -- o que os fez agir mesmo assim
5. VITÓRIA EXTERNA -- métricas objetivas agora
6. VITÓRIA INTERNA -- melhor momento desde então, o que conseguem fazer agora
```

### 8.5 Prioridade de Implementação (da mais rápida à mais elaborada)

1. **Competição** (grande influxo)
2. **Varredura de chats/mensagens** (já tem)
3. **Varredura de comunidade**
4. **Varredura de reviews**
5. **Varredura de redes sociais**
6. **Bonus unlocks**
7. **Award unlocks**
8. **Eventos**
9. **Lifecycle** (mais edição, maior qualidade)

### 8.6 Pergunta-Guia

> "Se eu fosse atropelado por um ônibus hoje, o que manteria o negócio crescendo amanhã?"

Uma empresa que depende do seu rosto é sempre menos valiosa. Coloque os clientes no holofote.

## SEÇÃO 9: PRINCÍPIOS FUNDAMENTAIS DA PUBLICIDADE

### 9.1 A Lição de Henry Ford

> "Você vai cansar da sua publicidade antes que seus prospects sequer saibam seu nome. Você faz campanhas para eles, não para você. Quando um anúncio finalmente funciona, continue rodando até que PARE de funcionar -- nunca até você ficar entediado."

### 9.2 Volume Nega Sorte

> "O processo é simples: faça muitos anúncios, identifique os poucos que performam, despeje 80% do esforço em remixar e refazer esses, gaste os outros 20% caçando o próximo vencedor. Um vencedor é suficiente para iniciar o flywheel. Permutações o mantêm girando."

### 9.3 Vencedores São Raros -- Esprema Cada Gota

> "80% das vendas vêm de 20% dos anúncios. Quando um anúncio finalmente clica, martele até morrer -- depois re-rode com variações do Kaleidoscope."

### 9.4 Frameworks São Agnósticos de Indústria

> "Essas estruturas de anúncio funcionam agnóstico de indústria. Todo hook, stack de prova e CTA pode vender o que quer que você tenha."

### 9.5 Orgânico Prediz Pago

> "Orgânico está sempre à frente de anúncios pagos por causa do volume e velocidade de testes dos criadores. Se quer ver onde anúncios estarão em 6-12 meses, olhe onde orgânico está hoje."

### 9.6 Otimize para LTV:CAC, Não CAC

> "Não otimize para CAC, otimize para LTV:CAC. Você pode conseguir clientes de R$500 que te pagam R$1.000, ou clientes de R$5.000 que te pagam R$50.000. Vão custar mais, mas você vai ganhar mais."

### 9.7 Especificidade É Tudo

> "Você demonstra sua expertise pelo quão específico consegue descrever o problema. Um problema tão específico que alguém ouve e diz 'ele deve ser um insider.'"

### 9.8 O Princípio Visual Avatar-First

Seu cliente ideal deve imediatamente ver o fundo, roupas e porta-voz como "igual a mim". Se mirando donos de academia, filme NUMA academia. Se mirando donos de agência, filme na frente de um quadro branco. "É tão simples, e mesmo assim tão poucas pessoas fazem."

### 9.9 Dois Tipos de Prova

1. **Resultados grandes** -- o que é possível (R$215K/mês em receita recorrente)
2. **Resultados medianos/modestos** -- o que é provável (R$23K/mês -- o mais baixo do leaderboard)

Sempre inclua ambos. "As pessoas querem saber que a coisa grande é possível, mas mais importante, querem entender o que é provável."

### 9.10 Admissões Danosas Constroem Confiança

Inclua disclaimers honestos nos anúncios: "Não é fácil, mas vale completamente a pena." A admissão aumenta credibilidade e diminui ceticismo.

### 9.11 O Princípio de Proximidade da Prova

```
Proximidade da prova à situação do prospect = Poder de persuasão

Mesma pessoa + mesma cidade + mesmo negócio + resultados visíveis >> Pessoa diferente + país diferente + afirmação verbal
```

### 9.12 As Duas Funções de Bons Anúncios

Bons anúncios fazem exatamente duas coisas:
1. **Capturam a atenção do prospect** (importância)
2. **Fazem o prospect agir** (benefício > custo de agir)

```
Atenção (hook) + Ação (oferta + CTA) = Bom Anúncio
```

### 9.13 A Equação de Valor nos Anúncios

Todo anúncio deve atingir os quatro pilares da equação de valor:

```
           Dream Outcome × Probabilidade Percebida de Conquista
Valor = ─────────────────────────────────────────────────────────
           Atraso de Tempo × Esforço & Sacrifício
```

- **Dream Outcome** -- o que o prospect deseja ardentemente
- **Probabilidade Percebida** -- prova de que vai funcionar para eles
- **Velocidade** -- quão rápido verão resultados
- **Facilidade** -- quão simples é o processo

## SEÇÃO 12: CONSELHOS TÁTICOS COMPLEMENTARES

### 12.1 Sobre Hooks de Anúncio

- Números no início de anúncios (especialmente números grandes) funcionam toda vez
- Quadros meio escritos invocam curiosidade (pessoas querem ver completo)
- Props que chamam SUA atenção vão chamar a de outros -- mantenha um armário de objetos interessantes
- Andar e falar quase sempre supera ficar parado

### 12.2 Sobre Prova

- Quanto mais a prova se aproxima da situação do prospect, menor o risco percebido
- Prova não-verbal (mostrar academia lotada) frequentemente vence afirmações verbais
- Timestamps aumentam credibilidade massivamente ("Entramos em março de 2023")
- Clipes não-verbais ganham mais alcance orgânico

### 12.3 Sobre Mirar Clientes Avançados

- Fale sobre coisas que só clientes avançados se importam (LTV, churn, margens)
- Use linguagem que só o escalão superior entenderia
- "Ninguém mais vai falar com o cliente avançado porque têm medo de ostracizar alguém. Então está maduro para a colheita."
- Você vai pegar tanto o mercado superior QUANTO o menor

### 12.4 Sobre Reciclagem de Hooks

- Reutilize hooks entre indústrias e negócios
- O hook "Bribe" apareceu tanto no Skool quanto no Lançamento do Livro -- produtos diferentes, ambos esmagaram
- Uma vez que encontrar um hook vencedor, faça permutações em formato estático também (imagens de hooks de vídeo)

### 12.5 Sobre Produção

- Anúncios não precisam ser fancy, precisam fazer pessoas clicarem
- Estilo selfie cria movimento extra e parece mais social-native
- Filme em cenários que seu avatar reconhece (academia para donos de academia, quadro branco para agências)
- Cenários "ADucacionais" (quadro branco, tela) funcionam melhor para produtos complexos/high-ticket
- Memes ou conteúdo meme-like atraem a maior porcentagem do público

### 12.6 Diretrizes de Escrita Hormozi

1. **Abaixo do nível de 3a série de leitura**
2. **Voz presente** (não progressiva)
3. **Ativa, não passiva**
4. **Evite advérbios** (geralmente significam verbos fracos)
5. **Frases simples/curtas** (geralmente uma vírgula máximo)
6. **Linguagem positiva** ("Continue" em vez de "Não Pare")
7. **Remova palavras redundantes**
8. **Remova palavras desnecessárias**

## SEÇÃO 14: CONEXÕES SISTÊMICAS

### 14.1 Conexão com `02_VALUE_EQUATION_ENGINE.md`

Todo anúncio deve atacar os 4 pilares da Equação de Valor. Ao criar hooks e meats, verifique se o anúncio endereça: Dream Outcome, Probabilidade Percebida, Velocidade e Facilidade. A Equação de Valor é o esqueleto invisível por trás de cada framework de anúncio.

### 14.2 Conexão com `03_OFFER_CREATION_SYSTEM.md`

A oferta é o que o anúncio vende. Um anúncio perfeito com uma oferta ruim fracassa. Antes de criar anúncios, a oferta deve estar validada pelo sistema de criação de ofertas. O CTA do anúncio aponta para a oferta -- eles são inseparáveis.

### 14.3 Conexão com `07_TESTING_OPTIMIZATION.md`

O Ad Kaleidoscope é uma instância do princípio geral de teste e otimização. Use os KPIs e Rácios de Ouro do módulo de testing para medir a performance dos anúncios. A Regra dos 100 se aplica diretamente: gaste R$500/dia em anúncios OU crie 100 minutos de conteúdo/dia.

### 14.4 Conexão com `01_FRAMEWORKS_OPERACIONAIS.md`

O princípio "Mais, Melhor, Novo" se aplica diretamente à publicidade:
- **Mais:** Mais anúncios, mais hooks, mais variações (Ad Assembly Line)
- **Melhor:** Kaleidoscope nos vencedores (Remix + Remake)
- **Novo:** 20% do esforço em ideias totalmente novas

### 14.5 Conexão com `06_CASE_LIBRARY_DENSE.md`

Os case studies deste módulo (Seção 10) complementam a biblioteca de cases geral. Use-os como exemplos concretos ao recomendar frameworks de anúncio para o usuário.

### 14.6 Conexão com `04_COMMUNICATION_DNA.md`

As diretrizes de escrita Hormozi (Seção 12.6) são uma extensão do DNA de comunicação. Ao criar copy para anúncios: abaixo do 3o ano de leitura, voz ativa, presente, sem advérbios, linguagem positiva.
