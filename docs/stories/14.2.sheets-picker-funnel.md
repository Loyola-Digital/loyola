# Story 14.2 — Picker de Planilhas no Funil

**Epic:** EPIC-14 Google Sheets — Pesquisas de Funil
**Status:** Draft
**Priority:** High
**Estimate:** M
**Depends on:** 14.1

---

## Story

Como usuario do Loyola X, quero selecionar uma planilha do Google Sheets e indicar quais abas sao de pesquisa, para vincular pesquisas ao meu funil.

## Acceptance Criteria

### AC1: UI — Botao vincular no funil
- [ ] Nova tab "Pesquisas" no funnel page (ao lado de Meta Ads e YouTube Ads)
- [ ] Botao "Vincular planilha" dentro da tab
- [ ] Se nao tem credencial Google conectada, mostrar aviso + link para settings

### AC2: UI — Picker de planilhas
- [ ] Dialog/popover que lista planilhas do usuario (nome + icone)
- [ ] Busca/filtro por nome
- [ ] Ao selecionar planilha, listar abas
- [ ] Checkbox em cada aba para marcar como "pesquisa"
- [ ] Botao "Vincular" salva as abas selecionadas

### AC3: UI — Lista de pesquisas vinculadas
- [ ] Mostrar planilhas vinculadas com badges das abas
- [ ] Botao remover (X) em cada pesquisa
- [ ] Suportar multiplas planilhas por funil

### AC4: Refresh token
- [ ] Usar refresh token do YouTube Channel ou Google Ads Account vinculado ao projeto
- [ ] Se nenhum conectado, pedir OAuth com scope sheets

## Tasks
