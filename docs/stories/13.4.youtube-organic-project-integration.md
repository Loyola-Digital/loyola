## Story

Como usuario do Loyola X, quero ver os dados organicos do YouTube dentro do meu projeto, no grupo Social da sidebar (ao lado do Instagram), para ter uma visao unificada de redes sociais organicas.

## Contexto Tecnico

Seguir o mesmo padrao do Instagram dentro de projetos:
- Sidebar: Social > Instagram + **YouTube**
- Pagina: `/projects/[id]/youtube-organic`
- Vinculacao: canal YouTube vinculado ao projeto via `youtube_channel_projects`

### Referencia (Instagram no projeto)

```
Sidebar:
  Social ▼
    Instagram    ← já existe
    YouTube      ← adicionar
```

Pagina `/projects/[id]/instagram` ja existe como referencia de UX.
