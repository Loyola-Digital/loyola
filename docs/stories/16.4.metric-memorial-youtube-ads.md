## Story

Como usuário do dashboard de YouTube Ads (Google Ads), quero ver o memorial de cálculo de cada KPI e ponto de gráfico, para entender a origem dos números de vídeo (CPV, VTR, completion rate etc.).

### Métricas a cobrir

| # | Métrica | Fórmula | Fonte |
|---|---------|---------|-------|
| 1 | Investimento | `Σ cost` | Google Ads API |
| 2 | Impressões | `Σ impressions` | Google Ads API |
| 3 | Visualizações | `Σ video_views` | Google Ads API |
| 4 | CPV | `Cost ÷ Views` | Derivado |
| 5 | VTR (View Rate) | `Views ÷ Impressions × 100` | Derivado |
| 6 | Completions 25% | `Σ video_quartile_p25_rate × views` | Google Ads API |
| 7 | Completions 50% | `... × views` | Google Ads API |
| 8 | Completions 75% | `... × views` | Google Ads API |
| 9 | Completions 100% | `... × views` | Google Ads API |
| 10 | Completion Rate | `Completions 100% ÷ Views × 100` | Derivado |
| 11 | Cliques | `Σ clicks` | Google Ads API |
| 12 | CTR | `Clicks ÷ Impressions × 100` | Derivado |
| 13 | Conversões | `Σ conversions` | Google Ads API |
| 14 | CPA | `Cost ÷ Conversions` | Derivado |
