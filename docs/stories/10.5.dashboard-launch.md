## Story

**As a** gestor de um lançamento,
**I want** a dedicated dashboard showing launch-specific metrics for the linked campaign,
**so that** I can track the launch performance across its phases in real time.

---

## Acceptance Criteria

### Page & Route

1. [ ] Create page at `/projects/:projectId/funnels/:funnelId`
2. [ ] Detect funnel type — if `launch`, render this dashboard. If `perpetual`, render Story 10.6.
3. [ ] Breadcrumb: Projetos > Expert Name > Funil Name
4. [ ] Period selector: custom date range (launch start → launch end) + presets (7d, 14d, 30d)

### KPI Cards (top row)

5. [ ] **Investido** — total spend da campanha no período
6. [ ] **Leads** — total de leads (se CRM integrado, senão mostra clicks como proxy)
7. [ ] **CPL** — Custo por Lead
8. [ ] **Vendas** — total de conversões
9. [ ] **Receita** — total de revenue (se disponível)
10. [ ] **ROAS** — Return on Ad Spend

### Gráfico de Spend Diário

11. [ ] AreaChart com spend diário da campanha
12. [ ] Eixo X: datas. Eixo Y: R$ investido
13. [ ] Tooltip com valor exato + data

### Funil de Conversão

14. [ ] Visualização de funil vertical (barras decrescentes):
    - Impressões → Cliques → Leads → Vendas
15. [ ] Mostrar taxa de conversão entre cada etapa
16. [ ] Cores: gradiente do dourado (brand) ao verde (conversão)

### Top Criativos do Lançamento

17. [ ] Grid de até 5 criativos com melhor CTR da campanha
18. [ ] Mostrar thumbnail, nome, spend, CTR, cliques
19. [ ] Reusa componentes da gallery existente (Story 8.x)

### Placement Breakdown

20. [ ] Mini tabela: onde os ads estão rodando (Feed, Stories, Reels, etc.)
21. [ ] Spend + impressões + CTR por placement
22. [ ] Filtrado pela campanha do funil

### States

23. [ ] **Loading**: Skeleton cards + skeleton chart enquanto dados carregam
24. [ ] **Empty state**: Se campanha não tem dados, mostrar mensagem "Campanha sem dados no período selecionado" com sugestão de trocar período
25. [ ] **No campaign linked**: Se funil não tem campanha vinculada, mostrar CTA para vincular
