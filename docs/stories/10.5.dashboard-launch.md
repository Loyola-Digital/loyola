## Story

**As a** gestor de um lançamento,
**I want** a dedicated dashboard showing launch-specific metrics for the linked campaign,
**so that** I can track the launch performance across its phases in real time.

---

## Acceptance Criteria

### Page & Route

1. [x] Create page at `/projects/:projectId/funnels/:funnelId`
2. [x] Detect funnel type — if `launch`, render this dashboard. If `perpetual`, render Story 10.6.
3. [x] Breadcrumb: Projetos > Expert Name > Funil Name
4. [x] Period selector: custom date range (launch start → launch end) + presets (7d, 14d, 30d)

### KPI Cards (top row)

5. [x] **Investido** — total spend da campanha no período
6. [x] **Leads** — total de leads (se CRM integrado, senão mostra clicks como proxy)
7. [x] **CPL** — Custo por Lead
8. [x] **Vendas** — total de conversões
9. [x] **Receita** — total de revenue (se disponível)
10. [x] **ROAS** — Return on Ad Spend

### Gráfico de Spend Diário

11. [x] AreaChart com spend diário da campanha

13. [x] Tooltip com valor exato + data

### Funil de Conversão

14. [x] Visualização de funil vertical (barras decrescentes):
    - Impressões → Cliques → Leads → Vendas
15. [x] Mostrar taxa de conversão entre cada etapa
16. [x] Cores: gradiente do dourado (brand) ao verde (conversão)

### Top Criativos do Lançamento

17. [x] Grid de até 5 criativos com melhor CTR da campanha
18. [x] Mostrar thumbnail, nome, spend, CTR, cliques
19. [x] Reusa componentes da gallery existente (Story 8.x)

### Placement Breakdown

20. [x] Mini tabela: onde os ads estão rodando (Feed, Stories, Reels, etc.)
21. [x] Spend + impressões + CTR por placement
22. [x] Filtrado pela campanha do funil

### States

23. [x] **Loading**: Skeleton cards + skeleton chart enquanto dados carregam
