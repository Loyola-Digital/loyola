## Story

Como usuario do Loyola X, quero digitar `/` no chat e ver uma lista de Minds disponiveis para selecionar, para facilmente consultar outro especialista durante a conversa.

## Contexto Tecnico

### Chat Input Atual

- `chat-input.tsx` — textarea com auto-expand + upload de arquivos
- `useChatStream.ts` — hook que envia mensagem via POST /api/chat
- Nao tem suporte a slash commands ou mentions

### Comportamento Desejado

1. Usuario digita `/` em qualquer posicao do texto
2. Popup aparece acima do cursor com lista de Minds
3. Digitando mais filtra a lista (ex: `/hor` → mostra "Hormozi")
4. Setas cima/baixo navegam, Enter seleciona
5. Ao selecionar, insere `/hormozi ` no texto (com espaco)
6. Popup fecha
7. ESC cancela o popup

### Dados

- Lista de Minds via `GET /api/minds` (ja existe)
- Cada Mind tem: id, name, squad, specialty, avatar
- Hook `useMinds()` ja existe no frontend

## Acceptance Criteria

### AC1: Deteccao do /
- [ ] Ao digitar `/` no textarea, abre popup de autocomplete
- [ ] So abre se `/` esta no inicio do texto ou apos um espaco
- [ ] Nao abre se `/` esta no meio de uma palavra

### AC2: Lista de Minds
- [ ] Popup mostra lista de Minds com: avatar, nome, squad
- [ ] Lista filtrada conforme usuario digita apos `/`
- [ ] Busca por nome (case insensitive)
- [ ] Max 8 resultados visiveis com scroll

### AC3: Navegacao
- [ ] Setas cima/baixo navegam a lista
- [ ] Enter seleciona o Mind destacado
- [ ] Click no item seleciona
- [ ] ESC fecha o popup
- [ ] Clicar fora fecha o popup

### AC4: Insercao
- [ ] Ao selecionar, substitui `/texto` por `/mindId ` no textarea
- [ ] Cursor posicionado apos o espaco
- [ ] Multiplas mencoes permitidas na mesma mensagem

### AC5: Visual
- [ ] Popup posicionado acima do input (nao sobre o texto)
- [ ] Estilo consistente com o resto do app (dark, rounded, border)
- [ ] Item selecionado com highlight
- [ ] Loading state enquanto carrega lista
