## Story

Como usuário do dashboard Instagram, quero ver ao passar o mouse sobre cada KPI e ponto dos gráficos o memorial de cálculo (fórmula + valores + fonte + período), para explicar números em reuniões sem precisar abrir o backend.

## Contexto Técnico

### Arquivos-alvo

- `packages/web/components/instagram/overview-cards.tsx` — KPI cards
- `packages/web/components/instagram/reach-chart.tsx` — gráfico de alcance
- `packages/web/components/instagram/audience-charts.tsx` — demographics
- `packages/web/components/instagram/posts-table.tsx` — métricas por post (se aplicável)

### Métricas a cobrir

| # | Métrica | Fórmula | Fonte |
|---|---------|---------|-------|
| 1 | Seguidores | (snapshot) | Instagram Graph API · `/me?fields=followers_count` |
| 2 | Saldo de Seguidores | `Follows − Unfollows` | Graph API · `follows_and_unfollows` |
| 3 | Alcance | `Σ reach diário` | Graph API · `account_insights reach` |
| 4 | Visualizações | `Σ views` | Graph API · `views` |
| 5 | Interações | `Σ total_interactions` | Graph API · `total_interactions` |
| 6 | Taxa de Engajamento | `Interações ÷ Alcance × 100` | Graph API · derivado |
| 7 | Salvamentos | `Σ saves` | Graph API |
| 8 | Compartilhamentos | `Σ shares` | Graph API |
| 9 | Cliques na Bio | `Σ profile_links_taps` | Graph API · `profile_links_taps` |
| 10 | Alcance diário (chart) | `reach do dia X` | Graph API · time series |

### Período

Todos os valores respeitam o filtro de data ativo (`period.since`/`period.until`). O memorial deve exibir o período formatado em pt-BR (ex: `20/03 — 17/06`).

## Acceptance Criteria

### AC1: Todos os 9 KPI cards têm memorial
- [ ] Cada card em `overview-cards.tsx` usa `<MetricWithTooltip>` com `formula` preenchido
- [ ] Fórmula simbólica correta (ver tabela acima)
- [ ] Valores individuais listados
- [ ] Fonte explícita (ex: "Instagram Graph API · total_interactions")

### AC2: Reach chart com memorial por ponto
- [ ] `reach-chart.tsx` usa `<FormulaChartTooltip>` em vez do tooltip default
- [ ] Cada ponto inclui `formula` no data item
- [ ] Hover em um dia mostra: "Alcance em 15/04: 42.300 (Instagram Graph API)"
- [ ] Se houver múltiplas séries (reach + impressions), cada uma tem memorial próprio

### AC3: Graceful fallback
- [ ] Se algum valor ainda não carregou (skeleton), card não mostra tooltip quebrado — omite `formula` prop
- [ ] Se a API não retorna `source` específico (ex: métrica derivada), usar a cadeia ("Alcance + Interações do período")

### AC4: Sem regressão visual
- [ ] Cards mantêm tamanho e layout atuais
- [ ] Filtro de período continua funcionando
- [ ] Valores numéricos formatados exatamente como antes

## Dev Notes

- **Helper centralizado:** criar `lib/formulas/instagram.ts` com uma factory por métrica para não repetir lógica de formatação
- `source` sugerido: sempre prefixar com "Instagram Graph API" + campo específico (ex: "Instagram Graph API · follows_and_unfollows")
- Formatação numérica pt-BR: usar `Intl.NumberFormat("pt-BR")` consistente
- Período: formatar com date-fns `format(date, "dd/MM")` + locale `ptBR`
