## Story

**As a** gestor de trafego da Loyola Digital,
**I want** the system to cross-reference sales data from Google Sheets with Meta Ads campaigns via UTM,
**so that** I can see real sales count, revenue, cost per sale, ROAS, and conversion rate per campaign/ad set/ad.

---

## Acceptance Criteria

1. [ ] Service `sales-analytics.ts`: le aba "sales" do Sheets via `getTabData()` com column mapping (utmCampaign, utmMedium, utmContent, valor)
2. [ ] Contagem de vendas por UTM — reusar padrao `countLeadsByUtm()` do traffic-analytics
3. [ ] Calculo de receita: somar coluna "valor" agrupada por UTM Campaign/Medium/Content
4. [ ] `GET /api/traffic/analytics/:projectId/campaigns` expandido — campos adicionais: `sales`, `revenue`, `costPerSale`, `roas`, `conversionRate`
5. [ ] `costPerSale` = spend / sales (null se sales = 0)
6. [ ] `roas` = revenue / spend (null se spend = 0)
7. [ ] `conversionRate` = sales / leads × 100 (null se leads = 0 ou null)
8. [ ] Mesma logica para rotas adsets e ads
9. [ ] Vendas sem UTM atribuivel mostradas em `unattributedSales` no response
10. [ ] Dashboard: colunas Vendas, Custo/Venda, ROAS visiveis quando aba "sales" mapeada
11. [ ] Cache compartilhado com traffic-analytics (mesma TTL 15min, mesmo `invalidateProjectCache`)
12. [ ] `pnpm typecheck` e `pnpm lint` passam sem erros

## Dev Notes

### Architecture Context

- **traffic-analytics.ts:** Ja tem `getLeadsForProject()`, `getMetaAccountForProject()`, `countLeadsByUtm()`, `normalizeUtm()`, `matchUtm()` — tudo reutilizavel
- **google-sheets.ts:** `getTabData(spreadsheetId, tabName, columnMapping)` — retorna rows com campos mapeados e UTMs normalizados
- **Tab mapping:** Aba "sales" mapeada com `{ utmCampaign, utmMedium, utmContent, valor }` — column mapping JSONB
- **CampaignAnalytics interface:** Atualmente tem `leads` e `cpl` — expandir com `sales`, `revenue`, `costPerSale`, `roas`, `conversionRate`
- **Cache:** `getCached()`, `setCache()`, `invalidateProjectCache()` ja existem

### Sales Data Flow

```
1. getTabData(spreadsheetId, "Vendas", columnMapping)
   → Returns: [{ utmCampaign: "campanha_1", utmMedium: "publico_frio", utmContent: "criativo_1", valor: "297" }, ...]

2. Group by UTM Campaign:
   → campanha_1: 15 vendas, receita R$ 4.455
   → campanha_2: 8 vendas, receita R$ 2.376

3. Cross with Meta Ads:
   → campanha_1: spend R$ 2.000, 15 vendas, R$ 4.455 receita
     → costPerSale: R$ 133.33
     → ROAS: 2.23 (receita / spend)
     → conversionRate: 15/50 × 100 = 30% (vendas / leads)
```
