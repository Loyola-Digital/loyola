## Story

Como usuario do Loyola X, quero ver metricas detalhadas de cada video do meu canal YouTube, incluindo retencao de audiencia e origens de trafego, para entender quais conteudos performam melhor organicamente.

## Contexto Tecnico

### YouTube Analytics API — Metricas por video

```
GET https://youtubeanalytics.googleapis.com/v2/reports
  ?ids=channel==MINE
  &startDate=2026-01-01
  &endDate=2026-04-07
  &metrics=views,estimatedMinutesWatched,likes,comments,shares,averageViewPercentage,subscribersGained
  &dimensions=video
  &sort=-views
  &maxResults=20
  &filters=video=={videoId}
```

### Origens de trafego

```
GET .../reports
  ?ids=channel==MINE
  &metrics=views,estimatedMinutesWatched
  &dimensions=insightTrafficSourceType
  &filters=video=={videoId}
```

### Retencao de audiencia (curva)

```
GET .../reports
  ?ids=channel==MINE
  &metrics=audienceWatchRatio
  &dimensions=elapsedVideoTimeRatio
  &filters=video=={videoId}
```

## Acceptance Criteria

### AC1: API — Metricas por video
- [ ] `GET /api/youtube-channels/:channelId/videos/:videoId/analytics?days=30`
- [ ] Retorna: views, watchTimeHours, likes, comments, shares, avgRetention, subscribersGained

### AC2: API — Origens de trafego
- [ ] `GET /api/youtube-channels/:channelId/videos/:videoId/traffic-sources?days=30`
- [ ] Retorna array: { source, views, watchTimeMinutes }
- [ ] Sources: SEARCH, SUGGESTED, EXTERNAL, BROWSE, etc

### AC3: API — Retencao de audiencia
- [ ] `GET /api/youtube-channels/:channelId/videos/:videoId/retention`
- [ ] Retorna array: { timePercent, watchPercent }
- [ ] timePercent: 0-100 (posicao no video), watchPercent: 0-1 (% audiencia retida)

### AC4: Frontend — Video detail modal/page
- [ ] Clique no video do top videos abre modal ou pagina de detalhe
- [ ] Player embed do YouTube
- [ ] KPI cards: Views, Watch Time, Likes, Comentarios, Shares, Retencao Media
- [ ] Grafico de retencao de audiencia (area chart)
- [ ] Origens de trafego (horizontal bar chart)

### AC5: Frontend — Comparativo de videos
- [ ] Tabela com todos os videos e metricas lado a lado
- [ ] Sortable por qualquer coluna
- [ ] Thumbnails dos videos

## Dev Notes

- `audienceWatchRatio` retorna a curva de retencao normalizada (0-1)
- `elapsedVideoTimeRatio` vai de 0.0 a 1.0 (posicao relativa no video)
- Traffic source types: YT_SEARCH, SUGGESTED, EXT_URL, BROWSE, NOTIFICATION, PLAYLIST, etc
- Labels amigaveis: YT_SEARCH → "Busca YouTube", SUGGESTED → "Sugeridos", EXT_URL → "Links externos"
- Cache de 30 min para analytics por video (dados nao mudam rapido)
