## Story

**As a** Loyola Digital employee,
**I want** a Settings page with organized tabs,
**so that** I can configure integrations like Meta/Instagram in a structured and intuitive interface.

---

## Acceptance Criteria

1. [ ] Nova rota `/settings` com layout de tabs (vertical sidebar ou horizontal tabs)
2. [ ] Tab "Geral" como placeholder (futuras configuracoes)
3. [ ] Tab "Meta / Instagram" como placeholder (sera preenchida em 3.3.2)
4. [ ] Link "Settings" adicionado na sidebar com icone Settings (Lucide `Settings`)
5. [ ] Breadcrumb: Settings > [Tab Name]
6. [ ] Responsivo: tabs viram dropdown/accordion no mobile
7. [ ] Protegida por auth (Clerk middleware — ja existe no layout)
8. [ ] Sidebar nav item com icone correto e estado ativo quando em /settings/*
