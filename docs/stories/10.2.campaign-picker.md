## Story

**As a** gestor criando um funil,
**I want** to see all campaigns from my connected Meta Ads account and pick one to link to the funnel,
**so that** the funnel dashboard loads only that campaign's data.

---

## Acceptance Criteria

### API

1. [x] `GET /api/projects/:projectId/meta-campaigns` — Returns list of campaigns from the Meta Ads account linked to the project
2. [x] Response: `{ campaigns: [{ id, name, status, objective }] }`
3. [x] Reuse existing `fetchCampaigns()` from `meta-ads.ts` service
4. [x] Cache response for 15 minutes (campaigns don't change often)
5. [x] If no Meta Ads account linked, return empty array with `{ campaigns: [], accountLinked: false }`

### Frontend Hook

6. [x] Create `useCampaignPicker(projectId)` hook in `use-funnels.ts`
7. [x] Returns `{ campaigns, isLoading, accountLinked }`
8. [x] `staleTime: 5 minutes`

### Campaign Selector Component

9. [x] Create `CampaignSelector` component: searchable dropdown with campaign name + status badge
10. [x] Show campaign objective as secondary text (e.g. "CONVERSIONS", "TRAFFIC")
11. [x] Filter out campaigns with status "DELETED"
12. [x] Empty state: "Conecte uma conta Meta Ads em Configurações" with link to `/settings/traffic`
