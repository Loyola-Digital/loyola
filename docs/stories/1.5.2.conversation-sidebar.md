## Story

**As a** authenticated Loyola Digital user chatting with a mind,
**I want** to see my conversation history in a sidebar and recent delegated tasks,
**so that** I can quickly switch between conversations and track my delegated work.

---

## Acceptance Criteria

1. [ ] Sidebar colapsavel no lado esquerdo da tela de chat
2. [ ] Lista de conversas do user (todas as minds) carregada via `GET /api/conversations`
3. [ ] Cada item mostra: mind avatar (MindAvatar), mind name, titulo da conversa, data relativa
4. [ ] Conversa ativa destacada com estilo visual diferenciado
5. [ ] Clicar em conversa navega para `/minds/{mindId}/chat?conversationId={id}`
6. [ ] Botao "Nova Conversa" no topo que navega para `/minds/{mindId}/chat` (sem conversationId)
7. [ ] Opcao de deletar conversa com confirmacao (dialog) — chama `DELETE /api/conversations/:id`
8. [ ] Secao "Tarefas Recentes" abaixo das conversas — carregada via `GET /api/tasks?limit=5`
9. [ ] Responsivo: sidebar como Sheet (drawer) no mobile, sidebar fixa no desktop
10. [ ] `pnpm typecheck` passa sem erros
11. [ ] `pnpm lint` passa sem erros
