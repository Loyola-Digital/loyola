## Overview

**BUG:** No gráfico "Leads: Reais vs Tendência vs Meta" (Story 18.22), as linhas de "Leads Reais" e "Tendência" estão plotando valores **diários** ao invés de **cumulativos**.

**Esperado:** Os valores devem ser acumulativos, assim como no gráfico "Leads Acumulados" (aba anterior).

**Impacto:** Os dados estão visualmente inconsistentes entre os dois gráficos, confundindo o usuário.

## Technical Notes

**Root Cause:**
Na função `expandChartData()`, os valores `leadsReais` e `tendencia` são calculados como **diários**:
```javascript
leadsReais: historyRow.leadsPagos + historyRow.leadsOrg + historyRow.leadsSemTrack,
tendencia: tendency[dayIndex] ?? 0,
```

**Solução:**
Converter para **cumulativo**:
```javascript
leadsReais: somaTodoOsDiasAnteroresInclusiveHoje,
tendencia: somaDoArrayTendencyAtéHoje,
```

Verificar se `calculateTendency()` retorna valores cumulativos ou diários e ajustar conforme necessário.
