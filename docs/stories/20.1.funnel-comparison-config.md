## Dev Notes

### Referência circular na FK
A tabela `funnels` auto-referencia com `compare_funnel_id`. Drizzle ORM suporta isso diretamente — usar a função `() => funnels.id` como referência (lazy reference) para evitar problema de hoisting no TypeScript.

Exemplo em schema.ts:
```ts
compareFunnelId: uuid("compare_funnel_id")
  .references((): AnyPgColumn => funnels.id, { onDelete: "set null" }),
```
Importar `AnyPgColumn` de `drizzle-orm/pg-core`.

### funnelShape — exemplo do campo adicional
```ts
compareFunnelId: f.compareFunnelId ?? null,
```

### updateFunnelSchema — campo adicional
```ts
compareFunnelId: z.string().uuid().nullable().optional(),
```

### PUT handler — campo adicional (conditional update)
```ts
if (body.compareFunnelId !== undefined) {
  updates.compareFunnelId = body.compareFunnelId;
}
```

### UI — Popover no header do funil
Usar `Popover` + `PopoverTrigger` + `PopoverContent` do shadcn/ui. Botão com ícone `Settings2` de lucide-react. Dentro: `Select` com `useFunnels(projectId)` filtrado — excluir o funil atual da lista.
