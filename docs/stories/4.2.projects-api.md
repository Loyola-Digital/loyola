```json

{ "name": "Projeto XYZ", "clientName": "Cliente XYZ Ltda", "description": "...", "color": "#d4a843" }
// Response 201
{ "id": "uuid", "name": "Projeto XYZ", "clientName": "Cliente XYZ Ltda", "color": "#d4a843", "isActive": true, "createdAt": "..." }
```

**GET /api/projects**
```json
[{ "id": "uuid", "name": "...", "clientName": "...", "color": "#...", "isActive": true, "createdAt": "..." }]
```

**GET /api/instagram/accounts?project_id=uuid**
```json
// Filtra accounts pelo project_id; sem ?project_id retorna todas do user
[{ "id": "uuid", "accountName": "...", "instagramUsername": "...", "projectId": "uuid" }]
```
