## Story

**As a** authenticated Loyola Digital user,
**I want** to see all my conversations in a dedicated page with filtering and pagination,
**so that** I can find and resume any past conversation with any mind.

---

## Acceptance Criteria

1. [x] Pagina `/conversations` com lista de todas as conversas do user
2. [x] Filtro por mind (select/dropdown com minds disponveis)
3. [x] Ordenacao por data (mais recente primeiro — `updated_at DESC`)
4. [x] Cada item mostra: mind avatar (MindAvatar), mind name, titulo da conversa, data relativa, message count badge
5. [x] Clicar navega para `/minds/{mindId}/chat?conversationId={id}`
6. [x] Paginacao com botoes "Anterior" / "Proximo" (offset-based)
7. [x] Empty state: "Nenhuma conversa ainda. Explore as minds!" com link para `/minds`
8. [x] Loading skeleton enquanto carrega dados
9. [x] `pnpm typecheck` passa sem erros
10. [x] `pnpm lint` passa sem erros
