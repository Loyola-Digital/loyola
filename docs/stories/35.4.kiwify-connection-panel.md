## 📖 Story

**Como** administrador de um projeto na Loyola Digital X,
**Quero** conectar a credencial da Kiwify (client_id + client_secret + account_id) na aba "Assinaturas" e ver o status da conexão,
**Para** habilitar o dashboard de recorrência da Kiwify do meu projeto — substituindo o placeholder "em breve" por integração real.

---

## ✅ Acceptance Criteria

1. **Hooks `lib/hooks/use-kiwify.ts`** (espelha `use-hotmart.ts`, React Query + `useApiClient`):
   - `useKiwifyConnection(projectId)` → GET connection.
   - `useSetKiwifyConnection(projectId)` → PUT `{ clientId, clientSecret, accountId }`; invalida `kiwify-connection` + `kiwify-products`.
   - `useDeleteKiwifyConnection(projectId)` → DELETE; invalida os mesmos.
   - `useKiwifyProducts(projectId, { months, enabled })` e `useKiwifyDashboard(projectId, { productId, months, enabled })`.
   - Tipos `KiwifyConnectionResponse`, `KiwifyProduct`, `KiwifyDashboard` (espelha o contrato do backend; campos null para gaps).

2. **`KiwifyConnectionPanel`** (`components/subscriptions/kiwify-connection-panel.tsx`, espelha `hotmart-connection-panel.tsx`):
   - Header "Kiwify — Assinaturas" + Badge "Conectado" quando conectado.
   - Estados loading/error/connected/form (admin) / aviso (não-admin).
   - **Form de conexão** (admin, não conectado): inputs `client_id` (text), `client_secret` (`type=password`), `account_id` (text). Validação não-vazio; toast sucesso/erro; **limpa o secret do estado local após sucesso**.
   - **Header conectado** com botão "Desconectar" (admin).
   - Texto de ajuda: onde gerar a credencial na Kiwify (Apps/API) e o que é o `account_id` (store-id).

3. **Substituição do stub:** o `KiwifyStub` em `hotmart-connection-panel.tsx` (ou na página) é **removido/substituído** pela renderização real de `KiwifyConnectionPanel`. A aba "Assinaturas" passa a mostrar Hotmart **e** Kiwify reais.

4. **Dashboard montado quando conectado:** quando `connected`, renderiza `<KiwifySubscriptionsDashboard />` (entregue na 35.5 — nesta story pode ser um placeholder importável que a 35.5 preenche, OU a 35.5 depende deste ponto de montagem). Documentar o ponto de integração.

5. **Gating de admin:** `isAdmin = role !== null && role !== "guest"` (mesmo padrão da página subscriptions). Não-admin vê dashboard (se conectado) mas não o form.

6. **Qualidade:** `npm run typecheck` + `npm run lint` (web) passam; sem erros de hidratação.
