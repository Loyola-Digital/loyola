## Story

Como usuario do Loyola X, quero conectar o Google Sheets via API para que o sistema possa ler planilhas de pesquisa e vincular a funis.

## Contexto Tecnico

### OAuth — Adicionar scope

Adicionar `https://www.googleapis.com/auth/spreadsheets.readonly` ao OAuth existente.
Pode ser adicionado junto ao flow do YouTube (mesmo consent) ou criar flow dedicado.

### Google Sheets API v4

- Base URL: `https://sheets.googleapis.com/v4/spreadsheets`
- Listar planilhas do usuario: Google Drive API `files.list` com mimeType `application/vnd.google-apps.spreadsheet`
- Listar abas: `GET /v4/spreadsheets/{spreadsheetId}` → retorna `sheets[].properties.title`
- Ler dados da aba: `GET /v4/spreadsheets/{spreadsheetId}/values/{sheetName}`

### Schema

Adicionar ao funil:
```typescript
surveys: jsonb("surveys").default([]).$type<{
  spreadsheetId: string;
  spreadsheetName: string;
  sheets: { sheetName: string; isSurvey: boolean }[];
}[]>()
```

Ou tabela separada `funnel_surveys`:
- id, funnel_id, spreadsheet_id, spreadsheet_name, sheet_name, is_survey, created_at

### Credenciais

Reaproveitar `GOOGLE_ADS_CLIENT_ID` + `GOOGLE_ADS_CLIENT_SECRET` (mesmo projeto Google Cloud).
O refresh token do Google Ads ou YouTube pode ja ter o scope necessario se foi pedido junto.
Caso contrario, pedir re-autorizacao com scope adicional.

## Acceptance Criteria

### AC1: OAuth scope
- [ ] Adicionar scope `spreadsheets.readonly` e `drive.readonly` ao OAuth do Google
- [ ] Funcionar com refresh token existente OU pedir re-auth com novos scopes

### AC2: Google Sheets Service
- [ ] `listSpreadsheets(accessToken)` — listar planilhas do usuario via Drive API

- [ ] Tabela `funnel_surveys`: id, funnel_id (FK), spreadsheet_id, spreadsheet_name, sheet_name, created_at
- [ ] Cascade delete com funil

## Dev Notes

- Google Drive API para listar planilhas: `GET https://www.googleapis.com/drive/v3/files?q=mimeType='application/vnd.google-apps.spreadsheet'&fields=files(id,name)`
- Google Sheets API para abas: `GET https://sheets.googleapis.com/v4/spreadsheets/{id}?fields=sheets.properties`
- Google Sheets API para dados: `GET https://sheets.googleapis.com/v4/spreadsheets/{id}/values/{sheetName}`
- O refresh token precisa ter scope `drive.readonly` para listar planilhas e `spreadsheets.readonly` para ler dados
- Se o usuario ja conectou YouTube/Google Ads mas sem scope de sheets, precisa de novo consent
- Cache de 5min para dados da planilha (pesquisas mudam devagar)
