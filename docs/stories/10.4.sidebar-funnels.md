## Story

**As a** usuário da plataforma,
**I want** to see my funnels listed inside each project folder in the sidebar,
**so that** I can navigate directly to any funnel dashboard with one click.

---

## Acceptance Criteria

### Sidebar Structure

1. [ ] Refatorar `ProjectFolder` para listar funis como sub-items abaixo dos módulos fixos:
   ```
   Expert A (projeto)
   ├── Instagram
   ├── Ads (global)
   ├── Conversas
   ├── ── separator ──
   ├── 🚀 Lançamento BF (funil tipo launch)
   ├── 🔄 Perpétuo Curso X (funil tipo perpetual)
   └── + Novo Funil (botão)
   ```
2. [ ] Ícone diferente por tipo: 🚀 (Rocket) para lançamento, Repeat para perpétuo
3. [ ] Clicar no funil navega para `/projects/:id/funnels/:funnelId`
4. [ ] Botão "+ Novo Funil" no final da lista — abre o Wizard (Story 10.3)
5. [ ] Active state no funil quando a rota atual é `/projects/:id/funnels/:funnelId`

### Data Fetching

6. [ ] Criar hook `useFunnels(projectId)` que retorna lista de funis do projeto
7. [ ] Cache com staleTime de 5 minutos
8. [ ] Loading state: 2 skeleton bars enquanto carrega

### Guest Sidebar

9. [ ] GuestSidebar também mostra funis dos projetos que o guest tem acesso
10. [ ] Guest não vê botão "+ Novo Funil" (apenas visualização)

### Mobile

11. [ ] Funis aparecem no Sheet drawer mobile (mesmo padrão da sidebar)
