## Story

**As a** gestor de trafego da Loyola Digital,
**I want** a unified dashboard that shows the complete funnel (Impressions → Clicks → Leads → Qualified Leads → Sales) with a visual funnel chart, a master table with all metrics, and project-level filtering,
**so that** I can evaluate the full performance of a launch at every level (campaign, ad set, ad) in one place.

### Sorting Implementation

```typescript
const [sortColumn, setSortColumn] = useState<string | null>(null);
const [sortDirection, setSortDirection] = useState<"asc" | "desc">("desc");

function handleSort(column: string) {
  if (sortColumn === column) {
    setSortDirection(prev => prev === "asc" ? "desc" : "asc");
  } else {
    setSortColumn(column);
    setSortDirection("desc");
  }
}

// Sort campaigns array before rendering
const sorted = [...campaigns].sort((a, b) => {
  if (!sortColumn) return 0;
  const aVal = a[sortColumn] ?? 0;
  const bVal = b[sortColumn] ?? 0;
  return sortDirection === "asc" ? aVal - bVal : bVal - aVal;
});
```
