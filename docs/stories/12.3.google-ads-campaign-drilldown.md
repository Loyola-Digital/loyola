## Story

Como usuario do Loyola X, quero ver uma tabela de campanhas do Google Ads com drill-down para grupos de anuncios e anuncios individuais, para analisar performance em cada nivel.

## Contexto Tecnico

Hierarquia do Google Ads: Campaign > Ad Group > Ad. Segue o mesmo padrao de drill-down do Meta Ads no `/traffic`.

### Queries GAQL

**Campanhas:**
```sql
SELECT
  campaign.id, campaign.name, campaign.status,
  metrics.cost_micros, metrics.impressions, metrics.clicks,
  metrics.video_views, metrics.ctr, metrics.average_cpc,
  metrics.average_cpm, metrics.conversions
FROM campaign
WHERE segments.date BETWEEN '{start}' AND '{end}'
  AND campaign.status != 'REMOVED'
```

**Ad Groups por campanha:**
```sql
SELECT
  ad_group.id, ad_group.name, ad_group.status,
  metrics.cost_micros, metrics.impressions, metrics.clicks,
  metrics.video_views, metrics.ctr, metrics.average_cpc
FROM ad_group
WHERE campaign.id = {campaignId}
  AND segments.date BETWEEN '{start}' AND '{end}'
```

**Ads por ad group:**
```sql
SELECT
  ad_group_ad.ad.id, ad_group_ad.ad.name, ad_group_ad.ad.type,
  ad_group_ad.ad.video_ad.video.id,
  metrics.cost_micros, metrics.impressions, metrics.clicks,
  metrics.video_views, metrics.ctr, metrics.average_cpc,
  metrics.video_quartile_p25_rate, metrics.video_quartile_p50_rate,
  metrics.video_quartile_p75_rate, metrics.video_quartile_p100_rate
FROM ad_group_ad
WHERE ad_group.id = {adGroupId}
  AND segments.date BETWEEN '{start}' AND '{end}'
```

## Acceptance Criteria

### AC1: API — Campaigns endpoint
- [ ] `GET /api/google-ads/analytics/:accountId/campaigns?days=30`
- [ ] Retorna array com: id, name, status, spend, impressions, clicks, views, cpv, viewRate, ctr, cpc, cpm, conversions
- [ ] Ordenado por spend desc

### AC2: API — Ad Groups endpoint
- [ ] `GET /api/google-ads/analytics/:accountId/adgroups?campaignId=X&days=30`
- [ ] Retorna array com metricas por ad group

### AC3: API — Ads endpoint
- [ ] `GET /api/google-ads/analytics/:accountId/ads?adGroupId=X&days=30`
- [ ] Retorna array com metricas por ad + video_id + retencao por quartil

### AC4: Frontend — Tabela de campanhas
- [ ] Tabela com colunas: Nome, Spend, Views, CPV, View Rate, Impr, CTR, CPC, Conversoes
- [ ] Sortable por qualquer coluna
- [ ] Highlight best/worst CPV e conversoes
- [ ] Clique expande para ad groups

### AC5: Frontend — Drill-down Ad Groups
- [ ] Nivel 2 indentado (igual Meta Ads)
- [ ] Clique expande para ads individuais

### AC6: Frontend — Drill-down Ads
- [ ] Nivel 3 com thumbnail do video (via YouTube Data API ou Google Ads)
- [ ] Sparkline de retencao (25% > 50% > 75% > 100%)
- [ ] Badge de tipo (Video, Display, etc)

## Dev Notes

- Video thumbnail do YouTube: `https://img.youtube.com/vi/{videoId}/maxresdefault.jpg`
- Fallback: `https://img.youtube.com/vi/{videoId}/hqdefault.jpg`
- video_id vem de `ad_group_ad.ad.video_ad.video.id` — precisa converter para YouTube video ID
- Google Ads video resource name: `customers/{customerId}/videos/{videoId}` — query `video.id` e `video.youtube_video_id`
