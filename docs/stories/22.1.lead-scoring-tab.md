### Scoring Algorithm

Pseudocode da funГ§ГЈo `computeBands(schema, { headers, rows })`:

```
questions = schema.scoring_model.questions
bands = schema.bands

# Mapa: questionId -> coluna index na planilha
questionColMap = {}
for q in questions:
  idx = headers.findIndex(h => h.trim().toLowerCase() === q.new_survey_column.trim().toLowerCase())
  questionColMap[q.id] = idx  # -1 se nГЈo achou

bandCounts = {bandId: 0 for band in bands}
unclassified = 0

for row in rows:
  totalScore = 0
  q4Idx = questionColMap.get("Q4")
  q4Value = q4Idx !== -1 ? row[q4Idx]?.trim() : ""
  q4Filled = q4Value !== "" && q4Value !== "(sem resposta)"

  for q in questions:
    colIdx = questionColMap[q.id]
    if colIdx === -1:
      totalScore += q.unmapped_default ?? 0
      continue
    answer = row[colIdx]?.trim() ?? ""
    match = q.answers.find(a => a.value.trim().toLowerCase() === answer.toLowerCase())
    if match:
      if match.points_conditional:  # Q3 EmpresГЎrio
        rule = match.points_conditional
        totalScore += q4Filled ? rule.if_q4_filled : rule.if_q4_empty
      else:
        totalScore += match.points
    else:
      totalScore += q.unmapped_default ?? 0

  # Classificar (range half-open: min <= score < max, exceto Гєltima banda que inclui max)
  band = bands.find(b => totalScore >= b.range.min && totalScore < b.range.max)
  if !band:
    # Tenta a Гєltima banda inclusiva (score == max permitido)
    band = bands.find(b => totalScore === b.range.max)
  if band:
    bandCounts[band.id] += 1
  else:
    unclassified += 1
```
