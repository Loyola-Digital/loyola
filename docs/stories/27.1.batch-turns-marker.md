## User Story

Como gestor de tráfego do Loyola X, quero marcar manualmente datas onde houve virada de lote (subida de preço, fim de bônus, etc.) na tabela "Dados diários" do dashboard de funil, para ler os números do funil com o contexto certo de quando o ticket/oferta mudou.

---

## Contexto Técnico

### Onde marcar
Tabela `CrossedFunnelDailyTable` (`packages/web/components/funnels/crossed-funnel-daily-table.tsx`) — usada no `LaunchDashboard` e no `meta-ads-spreadsheet-tab`.

### UX
- Clique direito numa linha de data → menu "Marcar virada de lote" → prompt do label
- Linha já marcada → clique direito → "Editar label" / "Remover virada"
- Visual: 📦 ao lado da data + borda esquerda âmbar na `<TableRow>`
- Tooltip no ícone mostra o label

### Storage
Nova tabela `funnel_batch_turns` escopada por `funnel_id`. Múltiplas viradas por funil, máx 1 por (funnel_id, date).

### Schema novo

```sql
CREATE TABLE funnel_batch_turns (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  funnel_id UUID NOT NULL REFERENCES funnels(id) ON DELETE CASCADE,
  date DATE NOT NULL,
  label VARCHAR(255) NOT NULL,
  created_by UUID REFERENCES users(id) ON DELETE SET NULL,
  created_at TIMESTAMPTZ DEFAULT NOW() NOT NULL,
  updated_at TIMESTAMPTZ DEFAULT NOW() NOT NULL,
  UNIQUE (funnel_id, date)
);

CREATE INDEX idx_batch_turns_funnel ON funnel_batch_turns (funnel_id);
```

### API
- `GET    /api/projects/:projectId/funnels/:funnelId/batch-turns` — lista
- `POST   /api/projects/:projectId/funnels/:funnelId/batch-turns` — `{ date, label }`
- `PATCH  /api/projects/:projectId/funnels/:funnelId/batch-turns/:id` — `{ label }`
- `DELETE /api/projects/:projectId/funnels/:funnelId/batch-turns/:id`

Guests bloqueados em mutações (igual padrão das outras rotas).

---

## Acceptance Criteria

1. **AC1 — Marcar:** Clique direito em linha de data sem virada → opção "Marcar virada de lote" → prompt do label → após confirmar, ícone 📦 aparece + linha ganha borda âmbar.
2. **AC2 — Editar:** Clique direito em linha já marcada → "Editar label" → prompt pré-preenchido → atualiza tooltip.
3. **AC3 — Remover:** Clique direito em linha já marcada → "Remover virada" → linha volta ao estado normal.
4. **AC4 — Persistência:** Marcações sobrevivem a refresh e são compartilhadas entre usuários do mesmo funil.
5. **AC5 — Tooltip:** Hover no ícone 📦 mostra o label customizado.
6. **AC6 — Linha "Total":** Não aceita virada de lote (só linhas de data).
