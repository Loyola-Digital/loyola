# Story 19.4 — Stage Type: Pago / Gratuito

**Epic:** 19 — Funnel Stages  
**Status:** Ready  
**Points:** 3

---

## User Story

Como usuário, ao criar ou editar uma etapa de funil, quero definir se ela é de **captação paga** ou **captação gratuita**, para que o sistema saiba quais funcionalidades financeiras habilitar naquela etapa.

---

## Acceptance Criteria

- [ ] AC1: Campo `stage_type` (`'paid'` | `'free'`, default `'free'`) adicionado à tabela `funnel_stages` via migration
- [ ] AC2: API retorna `stageType` no shape da etapa; `PUT /stages/:stageId` aceita `stageType` no body
- [ ] AC3: Dialog "Nova Etapa" exibe seletor de tipo (Captação Gratuita / Captação Paga) antes de confirmar
- [ ] AC4: `StageCard` (3 pontos) exibe opção "Alterar para Paga" / "Alterar para Gratuita" conforme tipo atual
- [ ] AC5: Sheet "Configurar Etapa" exibe o tipo atual e permite alterá-lo
- [ ] AC6: `FunnelStage` no `@loyola-x/shared` inclui campo `stageType: 'paid' | 'free'`

---
