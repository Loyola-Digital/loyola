## 📖 User Story

Como gestor de tráfego, quero ver um gráfico de tendência de leads que combine:
1. Uma projeção cinza (tendência) baseada na média de entrada diária de leads pagos + orgânicos
2. Uma área vermelha cumulativa mostrando a meta diária de leads que preciso capturar

para que eu possa acompanhar se estou no caminho certo para atingir a meta final de leads até o fim do lançamento pago.

## 🎬 Implementation Notes

### Fórmulas Detalhadas

#### 1. Média Diária (Pago + Orgânico)
```
dataPrimeiraCaptacao = primeiro dia com dados ≠ null
diasDecorridos = hoje - dataPrimeiraCaptacao + 1
totalLeadsPago = soma(leads_pago histórico)
totalLeadsOrganico = soma(leads_organico histórico)
totalLeadsPagoOrganicoHistorico = totalLeadsPago + totalLeadsOrganico

mediaDiaria = totalLeadsPagoOrganicoHistorico / diasDecorridos
```

#### 2. Projeção de Tendência (Cinza)
```
dataAtual = hoje
dataFinal = input do usuário
diasRestantes = dataFinal - dataAtual

para cada dia D em [dataAtual, dataFinal]:
  diasDesdeInicio = D - dataPrimeiraCaptacao + 1
  leadsAcumuladosHoje = valor real até hoje
  diasAProjetar = diasDesdeInicio - diasDecorridos
  
  if D <= dataAtual:
    tendenciaD = leadsAcumuladosHoje (valor real)
  else:
    tendenciaD = leadsAcumuladosHoje + (mediaDiaria × diasAProjetar)
```

#### 3. Meta Cumulativa (Vermelho)
```
periodoCapt acao = dataFinal - dataPrimeiraCaptacao + 1
metaDiaria = metaTotal / periodoCapt acao

para cada dia D em [dataPrimeiraCaptacao, dataFinal]:
  diasDesdeInicio = D - dataPrimeiraCaptacao + 1
  metaD = metaDiaria × diasDesdeInicio
```

### Exemplo Numérico
```
Dados Históricos (05/01 a 05/15 — 15 dias):
- Pago: 450 leads
- Orgânico: 150 leads
- Total: 600 leads acumulados até 05/15

Inputs do Usuário:
- Data Final: 30/06/2026
- Meta Total: 3.000 leads

Cálculos:
- Data Primeira Captura: 01/05
- Data Atual: 15/05
- Dias Decorridos: 15
- Média Diária = (450 + 150) / 15 = 40 leads/dia
- Período Captação = 30/06 - 01/05 + 1 = 61 dias
- Meta Diária = 3.000 / 61 ≈ 49,2 leads/dia

Projeção (16/05 — 30/06):
- 16/05: Tendência = 600 + (40 × 1) = 640 leads; Meta = 49,2 × 16 = 787 leads
- 17/05: Tendência = 600 + (40 × 2) = 680 leads; Meta = 49,2 × 17 = 837 leads
- ...
- 30/06: Tendência = 600 + (40 × 47) = 2.480 leads; Meta = 49,2 × 61 = 3.000 leads
```
