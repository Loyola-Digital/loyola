## Story

**As a** gestor de trafego da Loyola Digital,
**I want** to see performance metrics broken down by ad placement (Feed, Stories, Reels, Explore, etc.),
**so that** I can understand which placements deliver the best results and optimize budget allocation.

---

### Meta API — Breakdown Endpoint

```
GET /act_{id}/insights?breakdowns=publisher_platform,platform_position&fields=spend,impressions,clicks,ctr,cpc,cpm&date_preset=last_30d
```

Retorna dados como:
```json
{
  "data": [
    { "publisher_platform": "facebook", "platform_position": "feed", "spend": "120.50", "impressions": "5000", ... },
    { "publisher_platform": "instagram", "platform_position": "story", "spend": "80.30", "impressions": "3200", ... },
    { "publisher_platform": "instagram", "platform_position": "reels", "spend": "45.20", "impressions": "1800", ... }
  ]
}
```
