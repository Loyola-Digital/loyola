## Story

**As a** Loyola Digital admin/manager,
**I want** a Traffic section in the sidebar, a settings page to manage Meta Ads accounts, and a traffic dashboard page,
**so that** I can add/remove ad accounts, link them to projects, and access traffic data from the platform.

---

## Acceptance Criteria

1. [ ] Item "Trafego" na sidebar global com icone `TrendingUp`, posicionado entre "Instagram" e "Settings" no array `navItems`
2. [ ] Aba "Trafego" em `/settings/traffic` visivel apenas para admin/manager (seguir padrao de `ADMIN_TABS` no settings layout)
3. [ ] Settings page: formulario de cadastro com campos `Nome da conta`, `Ad Account ID` (placeholder: `act_XXXXXXX`), `Access Token`; botao "Conectar conta" com loading state
4. [ ] Settings page: lista de contas cadastradas mostrando nome, Ad Account ID, status (ativo/inativo), data de criacao; acoes: excluir (com confirmacao)
5. [ ] Settings page: para cada conta, exibir projetos vinculados com opcao de vincular/desvincular (dropdown de projetos disponiveis)
6. [ ] Pagina `/traffic` com listagem de contas do usuario mostrando nome e Ad Account ID; cada conta e um card clicavel (futuro: link para detalhes de campanhas)
7. [ ] Hook `useMetaAdsAccounts()` — lista contas via `GET /api/meta-ads/accounts`; com React Query
8. [ ] Hook `useCreateMetaAdsAccount()` — mutation para `POST /api/meta-ads/accounts`; com invalidation
9. [ ] Hook `useDeleteMetaAdsAccount()` — mutation para `DELETE /api/meta-ads/accounts/:id`
10. [ ] Estados visuais corretos: loading (skeleton), empty state ("Nenhuma conta de anuncios cadastrada"), erro de token invalido (mensagem vermelha)
11. [ ] `pnpm typecheck` e `pnpm lint` passam sem erros

### Hook Pattern (use-meta-ads.ts)

```typescript
// Seguir padrao de use-audit.ts
export function useMetaAdsAccounts() {
  const apiClient = useApiClient();
  return useQuery({
    queryKey: ["meta-ads-accounts"],
    queryFn: () => apiClient<MetaAdsAccount[]>("/api/meta-ads/accounts"),
  });
}

export function useCreateMetaAdsAccount() {
  const apiClient = useApiClient();
  const queryClient = useQueryClient();
  return useMutation({
    mutationFn: (data: CreateMetaAdsAccountInput) =>
      apiClient<MetaAdsAccount>("/api/meta-ads/accounts", { method: "POST", body: data }),
    onSuccess: () => queryClient.invalidateQueries({ queryKey: ["meta-ads-accounts"] }),
  });
}
```
