## Story

**As a** authenticated Loyola Digital user,
**I want** to view a detailed profile of a mind including bio, frameworks, and communication style,
**so that** I can understand the mind's expertise before starting a conversation.

---

## Acceptance Criteria

1. [ ] Pagina `/minds/[mindId]` com perfil completo da mind
2. [ ] Header: MindAvatar (lg), nome, squad name, especialidade
3. [ ] Secao "Sobre" com bio extraida dos artifacts
4. [ ] Secao "Frameworks" com lista de frameworks que a mind usa
5. [ ] Secao "Estilo de Comunicacao" com tone e vocabulario
6. [ ] Secao "Estatisticas" com contagem de artifacts, heuristics e conversas do user
7. [ ] Botao "Iniciar Conversa" que navega para `/minds/[mindId]/chat`
8. [ ] Loading state com skeleton
9. [ ] Error state (mind nao encontrada → 404 page ou mensagem)
10. [ ] Breadcrumb: Minds > Mind Name
11. [ ] `pnpm typecheck` passa sem erros
12. [ ] `pnpm lint` passa sem erros
