## Story

**As a** Loyola Digital employee,
**I want** a sidebar with collapsible project folders,
**so that** I can navigate per-client resources quickly without losing access to global views.

---

## Acceptance Criteria

1. [ ] Seção "Global" mantida no topo: Dashboard (`/`), Conversas (`/conversations`), Tasks (`/tasks`), Instagram (`/instagram`), Settings (`/settings`)
2. [ ] Separador visual entre seção Global e seção Projetos
3. [ ] Seção "Projetos": cada projeto é um folder colapsável com chevron toggle
4. [ ] Folder exibe cor do projeto como dot indicator + nome do projeto
5. [ ] Estado de colapso de cada projeto persistido em `localStorage` (key: `project-folder-{id}`)
6. [ ] Subitens visíveis quando expandido: 📊 Instagram → `/projects/[id]/instagram`, 💬 Conversas → `/projects/[id]/conversations`
7. [ ] Item ativo destacado (variant `secondary`)
8. [ ] Botão "+ Novo Projeto" ao fim da lista → abre `CreateProjectDialog`
9. [ ] `CreateProjectDialog`: campos `name`, `clientName`, `color` (color picker simples — 6 cores predefinidas) + botão Criar
10. [ ] Loading skeleton enquanto projetos carregam
11. [ ] Empty state inline quando sem projetos: "Nenhum projeto. Crie o primeiro."
12. [ ] Responsivo: comportamento de sheet no mobile mantido
13. [ ] Rota `/projects/[id]` criada — overview do projeto (nome, clientName, cor, lista de contas vinculadas)
14. [ ] `useProjects()` hook TanStack Query para listar projetos

## Dev Notes

### Architecture Context

- **Sidebar atual:** `packages/web/components/layout/app-sidebar.tsx` — refatorar mantendo comportamento desktop/mobile [Source: packages/web/components/layout/app-sidebar.tsx]
- **State:** Zustand `ui-store` para sidebarOpen — manter; adicionar localStorage para collapse de folders
- **Components:** shadcn/ui `Collapsible` para folders (verificar se já instalado, se não: `npx shadcn@latest add collapsible`)
- **API Client:** padrão `useApiClient()` + TanStack Query — igual a `use-instagram-accounts.ts`

### UI Design Reference

```
[Global]              ← label de seção
  🏠 Dashboard
  💬 Conversas
  ✅ Tasks
  📊 Instagram
  ⚙️  Settings
──────────────        ← Separator
[Projetos]            ← label de seção
  ▶ ● Cliente XYZ    ← chevron + dot colorido + nome (colapsado)
  ▼ ● Cliente ABC    ← expandido
    📊 Instagram
    💬 Conversas
  + Novo Projeto
```

### Key Notes

- `Collapsible` do shadcn/ui já é dependência do projeto (instalado em EPIC-1) — verificar
- Color picker: 6 cores predefinidas como swatches (não input hex livre) para simplicidade
- `project-folder-{id}` no localStorage: `"open"` ou `"closed"`
