## Story

Como usuario do Loyola X, quero ver uma galeria visual dos criativos de video do YouTube Ads com thumbnails e player, para comparar performance visual dos anuncios.

## Contexto Tecnico

YouTube videos tem thumbnails de alta resolucao acessiveis publicamente via URL padrao. O player pode ser embutido via iframe do YouTube. Segue o mesmo padrao da galeria de criativos do Meta Ads.

### Thumbnails do YouTube

```
Alta resolucao: https://img.youtube.com/vi/{youtubeVideoId}/maxresdefault.jpg
HD:             https://img.youtube.com/vi/{youtubeVideoId}/hqdefault.jpg
Media:          https://img.youtube.com/vi/{youtubeVideoId}/mqdefault.jpg
Default:        https://img.youtube.com/vi/{youtubeVideoId}/default.jpg
```

### Player Embed

```html
<iframe src="https://www.youtube.com/embed/{youtubeVideoId}?autoplay=1" allowfullscreen />
```

## Dev Notes

- YouTube thumbnails sao publicas — nao precisa de API key
- `maxresdefault.jpg` pode nao existir para todos os videos — usar `hqdefault.jpg` como fallback
- YouTube embed: `https://www.youtube.com/embed/{videoId}?autoplay=1&rel=0`
- Para obter youtube_video_id via Google Ads API:
  ```sql
  SELECT video.id, video.youtube_video_id
  FROM video
  WHERE video.id = '{googleAdsVideoId}'
  ```
- Qualidade de thumbnail nao e problema aqui (diferente do Meta) — YouTube sempre tem boa resolucao
