## Story

**As a** gestor da Loyola Digital,
**I want** a simple step-by-step wizard to create a funnel inside my project,
**so that** I can quickly set up a launch or evergreen funnel linked to the right campaign.

---

## Acceptance Criteria

### Wizard Flow (3 Steps)

1. [ ] **Step 1 — Nome & Tipo**: Input para nome do funil + seleção de tipo com cards visuais:
   - Card "Lançamento" com ícone de foguete, descrição: "Para lançamentos com data de início e fim. Dashboard com fases e timeline."
   - Card "Perpétuo" com ícone de ciclo, descrição: "Para funis evergreen. Dashboard com tendências e métricas contínuas."
   - Seleção obrigatória, highlight no card selecionado
2. [ ] **Step 2 — Vincular Campanha**: Mostra `CampaignSelector` (Story 10.2) para escolher a campanha do Meta Ads
   - Opcional: usuário pode pular e vincular depois
   - Se não tem conta Meta Ads, mostrar aviso com link pra settings
3. [ ] **Step 3 — Confirmação**: Resumo visual com nome, tipo, campanha selecionada + botão "Criar Funil"
4. [ ] Ao confirmar, chama `POST /api/projects/:projectId/funnels` e redireciona para o dashboard do funil

### UI/UX

5. [ ] Wizard abre como Dialog/Modal fullscreen no mobile, overlay no desktop
6. [ ] Progress indicator (dots ou steps bar) mostrando passo atual
7. [ ] Botões "Voltar" e "Próximo" em cada step
8. [ ] Animação de transição entre steps (slide horizontal)
9. [ ] Botão de criar funil acessível na page do projeto (botão "+ Novo Funil")
10. [ ] Após criação, toast de sucesso e redirect para `/projects/:id/funnels/:funnelId`

### Validação

11. [ ] Nome obrigatório, mínimo 3 caracteres
12. [ ] Tipo obrigatório
13. [ ] Não permitir criar funil com campanha já vinculada a outro funil do mesmo projeto
