## Story

Como usuario do Loyola X, quero ver um dashboard com KPIs e grafico diario das minhas campanhas de YouTube Ads, para acompanhar a performance do investimento em video.

## Contexto Tecnico

Dashboard segue o mesmo padrao visual do Meta Ads (`/traffic`). Dados vem da Google Ads API via GAQL (Google Ads Query Language).

### Queries GAQL necessarias

**Overview (KPIs agregados):**
```sql
SELECT
  metrics.cost_micros,
  metrics.impressions,
  metrics.clicks,
  metrics.video_views,
  metrics.ctr,
  metrics.average_cpc,
  metrics.average_cpm,
  metrics.conversions,
  metrics.cost_per_conversion,
  metrics.video_quartile_p25_rate,
  metrics.video_quartile_p50_rate,
  metrics.video_quartile_p75_rate,
  metrics.video_quartile_p100_rate
FROM campaign
WHERE segments.date BETWEEN '{start}' AND '{end}'
  AND campaign.status = 'ENABLED'
```

**Daily insights:**
```sql
SELECT
  segments.date,
  metrics.cost_micros,
  metrics.impressions,
  metrics.clicks,
  metrics.video_views
FROM campaign
WHERE segments.date BETWEEN '{start}' AND '{end}'
  AND campaign.id = {campaignId}
ORDER BY segments.date
```

## Acceptance Criteria

### AC1: API — Overview endpoint
- [ ] `GET /api/google-ads/analytics/:accountId/overview?days=30`
- [ ] Retorna: totalSpend, totalViews, totalImpressions, totalClicks, cpv, viewRate, ctr, cpc, cpm, conversions, costPerConversion
- [ ] cost_micros convertido para reais (dividir por 1_000_000)
- [ ] Cache de 15 min

### AC2: API — Daily insights endpoint
- [ ] `GET /api/google-ads/analytics/:accountId/daily?days=30&campaignId=X`
- [ ] Retorna array: { date, spend, views, impressions, clicks }
- [ ] Suporta filtro por campaignId (opcional)

## Dev Notes

- Google Ads API usa `searchStream` para queries — retorna dados em streaming
- `cost_micros` esta em microunidades (1 real = 1_000_000 micros)
- View Rate = video_views / impressions * 100
- CPV = cost / video_views
- Customer ID sem hifens na API, com hifens na UI
