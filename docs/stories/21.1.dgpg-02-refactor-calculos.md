## Especificação

### Fonte de Dados — Leads

**Decisão:** Usar **sempre a contagem da aba da planilha**, não a API do Facebook.

| Tipo | Fonte | Campo |
|------|-------|-------|
| Total Leads | Planilha | Coluna de leads |
| Data Lead | Planilha | Coluna de data de criação |
| Email Lead | Planilha | Coluna de email |
| Telefone Lead | Planilha | Coluna de telefone |

### Cálculos — Fórmulas

#### Investimento
```
= SUM(Meta Ads API spend) no período
```
(Mesmo padrão do funil de referência)

#### Leads (Contagem)
```
= COUNT(Planilha[leads]) no período
```

#### Leads Pagos vs Orgânicos
```
Leads Pagos = COUNT(Planilha[origin = "paid"]) 
Leads Orgânicos = COUNT(Planilha[origin = "organic"])
Leads Sem origem = COUNT(Planilha[origin = null ou vazio])

Total = Pagos + Orgânicos + Sem origem
```

#### CPL Pago
```
= Investimento / Leads Pagos
```

#### CPC (Link Clicks)
```
= Investimento / Meta Ads API[Link Clicks]
```

#### CTR
```
= (Meta Ads API[Link Clicks] / Meta Ads API[Impressions]) × 100
```

#### CPM
```
= (Investimento / Meta Ads API[Impressions]) × 1000
```

#### Taxa de Resposta (Pesquisa)
```
= (Pesquisa[Total Respostas] / Leads Totais) × 100
```

#### Match Pesquisa x Leads
```
FOR EACH survey response:
  IF Email(response) matches Email(lead) case-insensitive:
    → MATCHED
  ELSE IF Last8Digits(Phone(response)) == Last8Digits(Phone(lead)):
    → MATCHED
  ELSE:
    → UNMATCHED

matchedCount = COUNT(MATCHED)
unmatchedCount = COUNT(UNMATCHED)
matchedLeadIds = SET(ids de leads com match)
```

**Normalização obrigatória antes do match:**
- Email: lowercase, trim
- Telefone: remover símbolos, extrair últimos 8 dígitos numéricos
