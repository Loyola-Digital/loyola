## Story

Como usuario do Loyola X, quero ver um dashboard com metricas gerais do meu canal do YouTube, para acompanhar crescimento organico, views e engajamento.

## Contexto Tecnico

Dados vem da YouTube Analytics API v2 via `reports.query`.

### Endpoint principal

```
GET https://youtubeanalytics.googleapis.com/v2/reports
  ?ids=channel==MINE
  &startDate=2026-01-01
  &endDate=2026-04-07
  &metrics=views,estimatedMinutesWatched,subscribersGained,subscribersLost,likes,comments,shares,averageViewPercentage
  &dimensions=day
  &sort=day
```

### Videos do canal

```
GET https://www.googleapis.com/youtube/v3/search
  ?channelId={channelId}
  &part=snippet
  &order=date
  &maxResults=20
  &type=video
```

## Acceptance Criteria

### AC1: API — Overview endpoint
- [ ] `GET /api/youtube-channels/:channelId/overview?days=30`
- [ ] Retorna: totalViews, watchTimeHours, subscribersGained, subscribersLost, netSubscribers, totalLikes, totalComments, totalShares, avgRetention
- [ ] estimatedMinutesWatched convertido para horas

### AC2: API — Daily insights
- [ ] `GET /api/youtube-channels/:channelId/daily?days=30`
- [ ] Retorna array: { date, views, watchTimeMinutes, subscribersGained }
- [ ] Ordenado por data

### AC3: API — Top videos
- [ ] `GET /api/youtube-channels/:channelId/videos?limit=10`
- [ ] Retorna: videoId, title, thumbnailUrl, publishedAt, viewCount, likeCount, commentCount
- [ ] Dados via YouTube Data API (search + videos.list)

### AC4: Frontend — KPI Cards
- [ ] 8 cards: Views, Watch Time, Inscritos (+/-), Likes, Comentarios, Shares, Retencao Media, Videos publicados
- [ ] Mesmo estilo visual do Meta/YouTube Ads
- [ ] Skeleton loading

### AC5: Frontend — Grafico Diario
- [ ] Line chart: Views + Inscritos diarios (dual Y axis)
- [ ] Fontes brancas, tooltip dark
- [ ] Period selector personalizado

### AC6: Frontend — Top Videos
- [ ] Grid de cards com thumbnail (YouTube URL publica), titulo, views, likes
- [ ] Clique abre video no YouTube (ou embed)
- [ ] Ordenado por views desc

### AC7: Frontend — Pagina
- [ ] Pagina `/youtube-organic` (global) ou integrada no projeto
- [ ] Select para escolher canal (se multiplos)
- [ ] Empty state quando nao tem canal conectado

## Dev Notes

- YouTube Analytics API usa `ids=channel==MINE` quando autenticado com o dono do canal
- Para canais de terceiros: `ids=channel=={channelId}`
- Watch time vem em minutos (estimatedMinutesWatched) — converter pra horas no frontend
- Thumbnails dos videos: `https://img.youtube.com/vi/{videoId}/hqdefault.jpg`
- `averageViewPercentage` vem como percentual (0-100)
- Quota do YouTube Data API: 10.000 unidades/dia — cachear agressivamente (30min+)
