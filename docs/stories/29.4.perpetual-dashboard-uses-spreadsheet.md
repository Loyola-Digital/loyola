# Story 29.4 — PerpetualDashboard usa planilha como fonte de vendas

**Epic:** 29
**Status:** Ready for Review
**Points:** 5
**Depende de:** 29.2, 29.3

## User Story

Como usuário de funil perpétuo com planilha conectada, quero ver os KPIs (Vendas, Receita, CAC, Margem, ROAS) e o gráfico "Investimento no Tempo" usando dados da minha planilha (source-of-truth) em vez do Pixel Meta. Quando a planilha não está conectada, dashboard segue funcionando com dados Meta como hoje (fallback gracioso).

## Acceptance Criteria

- [x] **AC1:** Quando `usePerpetualSpreadsheet` retorna planilha conectada **E** `usePerpetualSalesData` retorna `semDados=false`:
  - KPI **Vendas** = `salesData.totalVendas`
  - KPI **Receita** = `salesData.faturamentoBruto`
  - KPI **CAC** = `overview.totalSpend / salesData.totalVendas` (Meta spend ÷ planilha vendas)
  - KPI **Margem** = `salesData.faturamentoBruto − overview.totalSpend`
  - KPI **Margem %** = `(margem / faturamentoBruto) × 100`
  - KPI **ROAS** = `salesData.faturamentoBruto / overview.totalSpend`
- [x] **AC2:** Quando planilha não conectada → comportamento atual (dados Meta) preservado integralmente.
- [x] **AC3:** Cada KPI alterado mostra **badge visual** "📄 planilha" ou tooltip indicando que veio da planilha (sutil, na borda do card ou no tooltip de formula).
- [x] **AC4:** Gráfico "Investimento no Tempo": linha de Receita usa `usePerpetualSalesDataDaily.byDay` quando planilha conectada; Investimento continua Meta.
- [x] **AC5:** Gráfico "Margem no Tempo": margem diária = `salesData.byDay[date] - meta.spend[date]` quando planilha conectada.
- [x] **AC6:** Loading state respeita ambas as queries (overview + salesData) — não pisca valores.
- [x] **AC7:** Sem regressão: dashboards de funis sem planilha conectada continuam exatamente iguais.
- [x] **AC8:** Typecheck passa.

## Dev Notes

### Pattern: derive, don't fork

Não criar dois componentes (PerpetualDashboardWithSheet vs Original). Calcula `effectiveMetrics` derivado e usa no mesmo JSX. Reduz manutenção.

### Date matching no daily chart

`dailyData` do Meta usa `date_start` ISO (`"2026-05-15"`). `salesDataDaily.byDay` usa mesma chave (`YYYY-MM-DD` local). Match direto.

### Badge "📄 planilha"

```tsx
{usingSpreadsheet && (
  <span className="absolute top-1 right-1 text-[9px] text-emerald-400/70" title="Dado vindo da planilha conectada">📄</span>
)}
```
KpiCard precisa de `relative` no container pra absolute funcionar — checar se já tem.

### Sem regressão

Critério crítico — funis sem planilha NÃO podem mudar. Validar visualmente abrindo um funil sem planilha antes de marcar Done.
