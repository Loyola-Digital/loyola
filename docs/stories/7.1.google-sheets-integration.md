## Story

**As a** Loyola Digital admin/manager,
**I want** to connect a Google Sheets spreadsheet to a project, list its tabs, and map columns for leads/sales/survey data,
**so that** the platform can pull CRM data from the spreadsheet and cross-reference it with Meta Ads campaigns.

## Acceptance Criteria

1. [ ] Tabelas `google_sheets_connections` e `google_sheets_tab_mappings` criadas com migration Drizzle
2. [ ] Indice UNIQUE em `google_sheets_connections(project_id)` — uma planilha por projeto
3. [ ] Indice UNIQUE em `google_sheets_tab_mappings(connection_id, tab_name)`
4. [ ] Service `google-sheets.ts`: conectar via Service Account JSON, validar acesso a planilha, listar abas, ler dados de aba
5. [ ] `POST /api/google-sheets/connections` — conectar planilha: recebe `projectId` + `spreadsheetUrl`, valida acesso via API, extrai `spreadsheetId` e nome, retorna conexao criada com lista de abas
6. [ ] `GET /api/google-sheets/connections/:projectId` — retorna conexao do projeto com tab mappings
7. [ ] `DELETE /api/google-sheets/connections/:id` — remove conexao e tab mappings (CASCADE)
8. [ ] `PUT /api/google-sheets/connections/:id/tabs` — mapear abas: recebe array de `{ tabName, tabType, columnMapping }`; `tabType` e `'leads' | 'survey' | 'sales'`
9. [ ] `GET /api/google-sheets/connections/:id/tabs/:tabName/preview` — retorna primeiras 10 linhas da aba (header + 9 rows) para auxiliar mapeamento de colunas
10. [ ] `GET /api/google-sheets/connections/:id/tabs/:tabName/data` — retorna todos os dados da aba com column mapping aplicado (UTMs normalizados)
11. [ ] Rate limiting: 60 req/min para Google Sheets API
12. [ ] Env var `GOOGLE_SERVICE_ACCOUNT_KEY` — JSON stringified da Service Account key
13. [ ] Erro claro se planilha nao esta compartilhada com email da Service Account
14. [ ] Zod validation em todos os inputs
15. [ ] `pnpm typecheck` e `pnpm lint` passam sem erros

### Google Sheets API Reference

```typescript
// Package: googleapis (npm install googleapis)
import { google } from "googleapis";

// Auth via Service Account
const auth = new google.auth.GoogleAuth({
  credentials: JSON.parse(process.env.GOOGLE_SERVICE_ACCOUNT_KEY!),
  scopes: ["https://www.googleapis.com/auth/spreadsheets.readonly"],
});

const sheets = google.sheets({ version: "v4", auth });

// Listar abas:
const meta = await sheets.spreadsheets.get({ spreadsheetId });
const tabs = meta.data.sheets?.map(s => s.properties?.title) ?? [];

// Ler dados de uma aba:
const res = await sheets.spreadsheets.values.get({
  spreadsheetId,
  range: `'${tabName}'`,  // ou range especifico como 'Leads!A1:Z10'
});
const rows = res.data.values ?? [];
// rows[0] = headers, rows[1..n] = data
```

### Extrair spreadsheetId da URL

```typescript
// URL format: https://docs.google.com/spreadsheets/d/{SPREADSHEET_ID}/edit...
function extractSpreadsheetId(url: string): string | null {
  const match = url.match(/\/spreadsheets\/d\/([a-zA-Z0-9-_]+)/);
  return match?.[1] ?? null;
}
```

### Schema Drizzle — Novas Tabelas

```typescript
// ============================================================
// GOOGLE SHEETS TABLES (EPIC-7)
// ============================================================

export const googleSheetsConnections = pgTable(
  "google_sheets_connections",
  {
    id: uuid("id").defaultRandom().primaryKey(),
    projectId: uuid("project_id")
      .notNull()
      .references(() => projects.id, { onDelete: "cascade" }),
    spreadsheetId: varchar("spreadsheet_id", { length: 200 }).notNull(),
    spreadsheetUrl: text("spreadsheet_url").notNull(),
    spreadsheetName: varchar("spreadsheet_name", { length: 200 }).notNull(),
    isActive: boolean("is_active").notNull().default(true),
    createdBy: uuid("created_by")
      .notNull()
      .references(() => users.id, { onDelete: "restrict" }),
    createdAt: timestamp("created_at", { withTimezone: true })
      .defaultNow()
      .notNull(),
    updatedAt: timestamp("updated_at", { withTimezone: true })
      .defaultNow()
      .notNull(),
  },
  (table) => [
    uniqueIndex("uq_gsheets_project").on(table.projectId),
    index("idx_gsheets_created_by").on(table.createdBy),
  ]
);

export const googleSheetsTabMappings = pgTable(
  "google_sheets_tab_mappings",
  {
    id: uuid("id").defaultRandom().primaryKey(),
    connectionId: uuid("connection_id")
      .notNull()
      .references(() => googleSheetsConnections.id, { onDelete: "cascade" }),
    tabName: varchar("tab_name", { length: 200 }).notNull(),
    tabType: varchar("tab_type", { length: 50 }).notNull(), // 'leads' | 'survey' | 'sales'
    columnMapping: jsonb("column_mapping").notNull(),
    // columnMapping shape:
    // For leads:  { utmCampaign: "B", utmMedium: "C", utmContent: "D", email: "A" }
    // For survey: { utmCampaign: "B", renda: "E", sexo: "F", filhos: "G", religiao: "H" }
    // For sales:  { utmCampaign: "B", utmMedium: "C", utmContent: "D", valor: "E" }
    isActive: boolean("is_active").notNull().default(true),
    createdAt: timestamp("created_at", { withTimezone: true })
      .defaultNow()
      .notNull(),
    updatedAt: timestamp("updated_at", { withTimezone: true })
      .defaultNow()
      .notNull(),
  },
  (table) => [
    unique("uq_gsheets_tab_connection_name").on(table.connectionId, table.tabName),
    index("idx_gsheets_tab_connection").on(table.connectionId),
  ]
);
```
