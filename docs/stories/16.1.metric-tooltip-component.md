## Story

Como usuário dos dashboards do Loyola X, quero poder passar o mouse sobre qualquer métrica calculada e ver um memorial completo (fórmula + valores + fonte + período), para entender como cada número foi obtido sem precisar abrir planilhas ou código.

## Contexto Técnico

### Stack disponível

- `@radix-ui/react-tooltip` já instalado via shadcn/ui em `packages/web/components/ui/tooltip.tsx`
- `recharts` aceita `<Tooltip content={<CustomComp />} />`
- `lucide-react` para ícone `Info` / `HelpCircle`

### Tipo compartilhado

Criar `packages/web/lib/types/metric-formula.ts`:

```typescript
export interface MetricFormulaValue {
  /** Rótulo humano: "Vendas", "Inscrições" */
  label: string;
  /** Valor numérico ou pre-formatado ("R$ 1.200,50") */
  value: number | string;
  /** Fonte do dado: "Meta Ads", "Google Sheets — CRM", "Instagram Graph API" */
  source: string;
}

export interface MetricFormula {
  /** Expressão simbólica: "Vendas ÷ Inscrições", "Clicks × CPC" */
  expression: string;
  /** Valores que entraram no cálculo */
  values: MetricFormulaValue[];
  /** Resultado formatado — ex: "900 ÷ 1.000 = 0,90 = 90%" */
  result: string;
  /** Período considerado, se houver filtro de data ativo */
  period?: string;
  /** Nota adicional — ex: "Considerados apenas leads qualificados" */
  note?: string;
}
```

### Componentes

**A) `<MetricWithTooltip>`** em `packages/web/components/metrics/metric-with-tooltip.tsx`

API:
```tsx
<MetricWithTooltip
  label="Taxa de Conversão"
  value="90%"
  formula={{ expression, values, result, period }}
>
  {/* children opcional: permite embrulhar card completo */}
</MetricWithTooltip>
```

Comportamento visual:
- Ícone `Info` (pequeno, `h-3 w-3`) ao lado do valor quando `formula` fornecida
- Hover no card **inteiro** (não só no ícone) abre o tooltip
- Tooltip alinhado com `side="top"` default, offset razoável
- Usa `<FormulaTooltipContent>` internamente

**B) `<FormulaTooltipContent>`** em `packages/web/components/metrics/formula-tooltip-content.tsx`

Renderiza:
```
┌─────────────────────────────────────┐
│ Fórmula: {expression}               │
├─────────────────────────────────────┤
│ {label}:     {value}   {source}     │  ← uma linha por value
│ {label}:     {value}   {source}     │
├─────────────────────────────────────┤
│ Resultado: {result}                 │
│ Período: {period}       (se houver) │
│ Obs: {note}             (se houver) │
└─────────────────────────────────────┘
```

Estilo: fundo `bg-card` com `border`, padding confortável, texto `text-xs`. Largura mínima 280px, máxima 400px.

**C) `<FormulaChartTooltip>`** em `packages/web/components/metrics/formula-chart-tooltip.tsx`

Adaptador do Recharts — recebe `{ active, payload, label }` do Recharts e renderiza `<FormulaTooltipContent>` usando dados do `payload[0].payload.formula` (convenção: o gráfico preenche `formula` em cada ponto).

### Convenção de uso com Recharts

```tsx
const data = months.map(m => ({
  month: m.name,
  roas: m.roas,
  formula: {
    expression: "Receita ÷ Investimento",
    values: [...],
    result: `${m.roas}x`,
    period: m.dateRange,
  } satisfies MetricFormula,
}));

<LineChart data={data}>
  <Tooltip content={<FormulaChartTooltip />} />
  <Line dataKey="roas" />
</LineChart>
```

## Acceptance Criteria

### AC1: Tipo `MetricFormula` compartilhado
- [ ] Arquivo `packages/web/lib/types/metric-formula.ts` criado
- [ ] Exporta `MetricFormula` e `MetricFormulaValue`
- [ ] Campos opcionais (`period`, `note`, `source`) marcados corretamente

### AC2: `<MetricWithTooltip>` funcional
- [ ] Aceita `label`, `value`, `formula` + `children` opcional
- [ ] Quando `formula` ausente, renderiza sem ícone nem tooltip (graceful)
- [ ] Hover em qualquer parte do card abre tooltip (não só no ícone)
- [ ] Delay de 200ms antes de abrir (evita tooltip piscando)

### AC3: `<FormulaTooltipContent>` visual
- [ ] Renderiza fórmula simbólica no header com destaque visual
- [ ] Cada valor em linha própria: `label ........ value  (source)`
- [ ] `source` em cor `text-muted-foreground` (sutil)
- [ ] Resultado destacado no rodapé
- [ ] `period` e `note` só aparecem se fornecidos
- [ ] Largura min 280px / max 400px
- [ ] Quebra de linha em expressões/labels longos

### AC4: `<FormulaChartTooltip>` para Recharts
- [ ] Compatível com prop `content` do `<Tooltip>` do Recharts
- [ ] Lê `formula` de `payload[0].payload.formula`
- [ ] Se não houver `formula` no payload, faz fallback para tooltip default do Recharts (label + valor)
- [ ] Suporta múltiplas séries no mesmo ponto (empilha múltiplos memoriais se cada série tiver formula própria)

### AC5: Acessibilidade
- [ ] Tooltip abre também em foco via teclado (`Tab`)
- [ ] `aria-describedby` ligando o valor ao tooltip
- [ ] Texto do tooltip selecionável (para copiar)

### AC6: Storybook visual / exemplos
- [ ] Criar uma página de demonstração em `packages/web/app/(app)/dev/metrics-demo/page.tsx` com:
  - Card KPI com fórmula
  - Card KPI sem fórmula (fallback)
  - LineChart com memorial em cada ponto
  - BarChart com memorial
- [ ] Página acessível apenas em dev (admin)
