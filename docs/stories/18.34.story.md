## Story

Na tabela "Dados Diários" (Captação Paga), o tooltip da métrica "Total Ingressos" está ilegível — tudo em uma linha e sem contexto claro sobre o que cada valor representa.

**Current:** Tooltip mostra tudo junto, difícil de ler.  
**Target:** Tooltip formatado em múltiplas linhas com explicação clara de cada componente.

## Acceptance Criteria

### AC1: Formatar Tooltip "Total Ingressos"
- [ ] Tooltip separado em 3 linhas:
  ```
  Total Ingressos = Ingressos Pg + Ingressos Org + Ingressos s/ track
  Ingressos Pg = Ingressos que vieram de mídia paga
  Ingressos Org = Ingressos com origem orgânica
  ```
- [ ] Sem caracteres especiais ou de escape corrompidos
- [ ] Line breaks renderizam corretamente (HTML `<br/>` ou `\n` conforme contexto)

### AC2: Formatar Tooltip para Outros Campos
- [ ] Mesmo padrão para "Total Leads" (quando em Captação Gratuita)
  ```
  Total Leads = Leads Pg + Leads Org + Leads s/ track
  Leads Pg = Leads que vieram de mídia paga
  Leads Org = Leads com origem orgânica
  ```
- [ ] Tooltip "Faturamento Bruto" (se aplicável) também revisado

### AC3: Validação de Renderização
- [ ] Tooltip aparece corretamente ao hover
- [ ] Sem cortes ou truncamentos de texto
- [ ] Funciona em desktop e mobile (se houver tooltips em mobile)

### AC4: Testes
- [ ] Captura screenshot do tooltip completo
- [ ] Validar em diferentes browsers (Chrome, Firefox, Safari)
- [ ] Verificar que tooltips não quebram layout
