## Story

**As a** gestor de um funil perpétuo (evergreen),
**I want** a dashboard focused on trends and continuous optimization metrics,
**so that** I can monitor long-term performance and identify when costs are rising or ROAS is dropping.

---

## Acceptance Criteria

### Page & Route

1. [ ] Same route as launch: `/projects/:projectId/funnels/:funnelId`
2. [ ] If funnel type is `perpetual`, render this dashboard
3. [ ] Breadcrumb: Projetos > Expert Name > Funil Name
4. [ ] Period selector: 7d, 14d, 30d, 90d (padrão: 30d)

### KPI Cards (top row)

5. [ ] **Investido (período)** — total spend no período selecionado
6. [ ] **CAC** — Custo de Aquisição (spend / conversões)
7. [ ] **CPL** — Custo por Lead
8. [ ] **ROAS** — Return on Ad Spend
9. [ ] **Vendas** — total de conversões no período
10. [ ] Cada KPI mostra variação vs período anterior (badge +X% ou -X%)

### Gráfico de Tendência (dual axis)

11. [ ] LineChart com duas linhas: Spend (R$) e ROAS (x)
12. [ ] Eixo Y esquerdo: spend. Eixo Y direito: ROAS
13. [ ] Período diário no eixo X
14. [ ] Tooltip mostrando ambos valores
15. [ ] Linha de ROAS com destaque (mais grossa, cor diferente)

### CPL Trend

16. [ ] LineChart simples mostrando evolução do CPL diário
17. [ ] Linha de referência (média do período) como linha pontilhada
18. [ ] Cor verde quando abaixo da média, vermelha quando acima
19. [ ] Ajuda a identificar: "tá ficando mais caro adquirir lead?"

### Criativos Ativos — Performance Comparativa

20. [ ] Tabela/grid dos criativos ativos na campanha
21. [ ] Colunas: thumbnail, nome, spend, CTR, CPC, conversões
22. [ ] Ordenável por qualquer coluna
23. [ ] Highlight no criativo com melhor e pior performance

### Distribuição por Placement

24. [ ] Donut chart ou horizontal bars: spend por placement
25. [ ] Feed vs Stories vs Reels vs Audience Network
26. [ ] Mostra onde o dinheiro está indo

### States

27. [ ] **Loading**: Skeleton cards + skeleton charts enquanto dados carregam
28. [ ] **Empty state**: "Campanha sem dados no período" com sugestão de trocar período
29. [ ] **No campaign linked**: CTA para vincular campanha ao funil
30. [ ] **Nota**: CPL calculado como `spend / clicks` é proxy. CPL real requer integração CRM futura
