## Story

Como usuário do dashboard de tráfego (Meta Ads), quero ver o memorial de cálculo de cada métrica (ROAS, CPL, CPA, CTR, frequência etc.) e de cada ponto dos gráficos, para defender resultados de campanha em reuniões com clientes.

## Contexto Técnico

### Arquivos-alvo

- `packages/web/app/(app)/traffic/page.tsx` — dashboard principal
- `packages/web/app/(app)/projects/[id]/traffic/page.tsx` — dashboard por empresa
- `packages/web/components/funnels/**` — se houver gráficos reutilizáveis

### Métricas a cobrir

| # | Métrica | Fórmula | Fonte |
|---|---------|---------|-------|
| 1 | Investimento | `Σ spend` | Meta Ads API · `insights.spend` |
| 2 | Impressões | `Σ impressions` | Meta Ads API |
| 3 | Alcance | `Σ reach` | Meta Ads API |
| 4 | Cliques | `Σ clicks` | Meta Ads API |
| 5 | CTR | `Clicks ÷ Impressions × 100` | Derivado |
| 6 | CPC | `Spend ÷ Clicks` | Derivado |
| 7 | CPM | `Spend ÷ Impressions × 1000` | Derivado |
| 8 | Frequência | `Impressions ÷ Reach` | Derivado |
| 9 | Leads | `Σ actions[lead]` | Meta Ads API · `actions` |
| 10 | CPL | `Spend ÷ Leads` | Derivado |
| 11 | Compras | `Σ actions[purchase]` | Meta Ads API |
| 12 | CPA | `Spend ÷ Purchases` | Derivado |
| 13 | Receita | `Σ action_values[purchase]` | Meta Ads API |
| 14 | ROAS | `Revenue ÷ Spend` | Derivado |
| 15 | Gráfico spend/time | `spend do dia X` | Time series |

### Filtros

- Período ativo (date range)
- Contas Meta vinculadas à empresa
- Campanhas específicas (filter opcional)

O memorial deve incluir o escopo ativo (ex: `Período: 20/03 — 17/06, 3 campanhas selecionadas`).

## Acceptance Criteria

### AC1: KPI cards cobertos
- [ ] Todos os 14 KPI cards acima usam `<MetricWithTooltip>`
- [ ] Métricas derivadas (CTR, CPC, CPM, CPL, CPA, ROAS, Frequência) mostram os 2 operandos + operação
- [ ] Métricas brutas (impressions, clicks, spend) mostram apenas "Σ no período" + fonte

### AC2: Filtros no memorial
- [ ] Período formatado em pt-BR no `period`
- [ ] Se 1+ campanha filtrada, adicionar `note: "Considerando N campanha(s) selecionada(s)"`
- [ ] Se 1+ conta Meta vinculada, listar contas (ex: `Fonte: Meta Ads API · contas: Conta A, Conta B`)

### AC3: Gráficos
- [ ] Gráfico de spend ao longo do tempo → `<FormulaChartTooltip>`
- [ ] Gráfico de ROAS ao longo do tempo → `<FormulaChartTooltip>` com memorial de cada ponto mostrando spend/revenue do dia
- [ ] Gráfico de leads × conversões → idem
- [ ] Se existir ranking de criativos/campanhas (bar chart), hover em barra mostra memorial do valor

### AC4: Moeda e formatação
- [ ] Valores em BRL formatados: `R$ 1.234,56`
- [ ] Fórmula usa símbolo de moeda: `R$ 1.200 ÷ 40 = R$ 30`

### AC5: Sem regressão
- [ ] Filtro de período, conta e campanha continuam funcionais
- [ ] Performance: não degradação sensível na renderização dos cards
