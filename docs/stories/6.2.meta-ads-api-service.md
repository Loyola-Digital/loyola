## Story

**As a** Loyola Digital admin/manager,
**I want** API endpoints to create, list and delete Meta Ads accounts with validated tokens,
**so that** I can manage ad account connections and pull campaign data from the Marketing API.

---

## Acceptance Criteria

1. [ ] `POST /api/meta-ads/accounts` — cria conta com `accountName`, `metaAccountId`, `accessToken`; valida token via chamada `GET /act_{metaAccountId}?access_token={token}` a Marketing API antes de salvar; retorna 400 se token invalido
2. [ ] `GET /api/meta-ads/accounts` — lista contas; admin/manager ve todas, guest negado (403)
3. [ ] `DELETE /api/meta-ads/accounts/:id` — remove conta; admin/manager only
4. [ ] `POST /api/meta-ads/accounts/:id/projects/:projectId` — vincula conta a projeto; retorna 409 se ja vinculada
5. [ ] `DELETE /api/meta-ads/accounts/:id/projects/:projectId` — desvincula conta de projeto
6. [ ] `GET /api/meta-ads/accounts/:id/campaigns` — retorna campanhas da conta via Marketing API (`fields=id,name,status,objective,daily_budget,lifetime_budget`)
7. [ ] `GET /api/meta-ads/accounts/:id/insights?days=30` — retorna insights agregados: impressions, reach, clicks, spend, ctr, cpc, cpm, conversions
8. [ ] Token encriptado com AES-256-CBC (reusar `encryptToken`/`decryptToken` do servico Instagram)
9. [ ] Rate limiting: reusar padrao existente
10. [ ] Zod validation em todos os inputs
11. [ ] `pnpm typecheck` e `pnpm lint` passam sem erros

## Dev Notes

### Architecture Context

- **API Framework:** Fastify com `fastify-plugin` [Source: packages/api/src/routes/instagram.ts]
- **Validation:** Zod schemas para params e body [Source: padrao em routes/instagram.ts]
- **Encryption:** `packages/api/src/services/instagram.ts` — funcoes `encryptToken(token)` e `decryptToken(encrypted, iv)` com AES-256-CBC
- **Meta Marketing API v21.0:** Base URL `https://graph.facebook.com/v21.0`
  - Validar conta: `GET /act_{ID}?fields=name,account_status&access_token={TOKEN}`
  - Campanhas: `GET /act_{ID}/campaigns?fields=id,name,status,objective,daily_budget,lifetime_budget&access_token={TOKEN}`
  - Insights: `GET /act_{ID}/insights?fields=impressions,reach,clicks,spend,ctr,cpc,cpm,conversions&date_preset=last_30d&access_token={TOKEN}`
- **Drizzle ORM:** queries via `fastify.db` (registrado no plugin)
- **Schema imports:** `metaAdsAccounts`, `metaAdsAccountProjects`, `projects` de `../db/schema.js`
