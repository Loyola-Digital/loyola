# Story 19.5 — Planilhas de Vendas para Etapas Pagas

**Epic:** 19 — Funnel Stages  
**Status:** Ready  
**Points:** 5  
**Depende de:** Story 19.4

---

## User Story

Como usuário de uma etapa do tipo **paga**, quero conectar até 2 planilhas do Google Sheets na aba Planilhas — uma de **vendas de captação** e uma de **produto principal** — mapeando as colunas relevantes, para que o sistema saiba onde buscar os dados financeiros daquela etapa.

---

## Acceptance Criteria

- [ ] AC1: Nova tabela `stage_sales_spreadsheets` no DB com campos: `id`, `stage_id`, `subtype` (`'capture'|'main_product'`), `spreadsheet_id`, `spreadsheet_name`, `sheet_name`, `column_mapping` (jsonb), `created_at`
- [ ] AC2: API CRUD: `GET/POST/DELETE /api/projects/:projectId/funnels/:funnelId/stages/:stageId/sales-spreadsheets`
- [ ] AC3: Na aba **Planilhas** de uma etapa paga, exibir 2 seções exclusivas: "Captação" e "Produto Principal" (etapas gratuitas não exibem essas seções)
- [ ] AC4: Cada seção tem botão "Conectar planilha" que abre wizard de conexão (igual ao existente em `FunnelSpreadsheetWizardDialog`) com seleção de spreadsheet → sheet → mapeamento de colunas
- [ ] AC5: Colunas mapeáveis: `email`, `valorBruto`, `valorLiquido`, `formaPagamento`, `canalOrigem` — todos opcionais exceto `email` (obrigatório para deduplicação)
- [ ] AC6: Após conexão, exibir card com nome da planilha + sheet + botão de desconectar
- [ ] AC7: Somente 1 planilha por `subtype` por etapa (upsert — reconectar substitui a anterior)
- [ ] AC8: Hook `useStageSalesSpreadsheets(projectId, funnelId, stageId)` no web
