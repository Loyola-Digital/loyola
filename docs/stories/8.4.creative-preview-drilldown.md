## Story

**As a** gestor de trafego da Loyola Digital,
**I want** to see the creative thumbnail next to each ad in the drill-down table and click it to see the full creative,
**so that** I can identify which creatives are performing at every level of the campaign hierarchy.

---

## Acceptance Criteria

1. [ ] Na linha de Ad (nivel 2 do drill-down), adicionar mini-thumbnail (40x40px) antes do nome do ad
2. [ ] Thumbnail clicavel: abre modal/lightbox com imagem full + titulo + body do criativo
3. [ ] Se video: modal mostra thumbnail + link "Ver no Meta" (link para o ad no Business Manager, se disponivel)
4. [ ] Badge de tipo (Imagem/Video/Carousel) na linha do ad
5. [ ] Layout da linha nao quebra com thumbnail — imagem alinhada com texto do nome
6. [ ] Fallback: se sem criativo, mostra icone generico pequeno
7. [ ] Lightbox com botao de fechar, click fora fecha, ESC fecha
8. [ ] `pnpm typecheck` e `pnpm lint` passam

## Dev Notes

### Architecture Context

- **DrillDownAds:** `packages/web/app/(app)/traffic/page.tsx` (linha ~477) — componente que renderiza ads dentro de adsets
- **DrillDownRow:** Componente generico que renderiza linha de adset ou ad (nivel 1 ou 2)
- **Ad data:** Apos Story 8.1, `useTrafficAds()` retornara `creative: MetaAdCreative | null` em cada ad
- **UI libs:** shadcn Dialog (para lightbox), Lucide icons

### Layout da Linha com Thumbnail

```
ANTES:                                 DEPOIS:
──────                                 ───────
              Ad Name                  [IMG] Ad Name [Video]
              R$500  5K  200  ...             R$500  5K  200  ...
```

```typescript
// Na coluna "Nome" do DrillDownRow (level === 2):
<td className="py-1.5 px-3 text-[11px] pl-14">
  <span className="inline-flex items-center gap-2">
    {item.creative?.thumbnailUrl ? (
      <img
        src={item.creative.thumbnailUrl}
        className="w-10 h-10 rounded object-cover cursor-pointer"
        onClick={(e) => { e.stopPropagation(); openLightbox(item.creative); }}
      />
    ) : (
      <div className="w-10 h-10 rounded bg-muted flex items-center justify-center">
        <ImageIcon className="h-4 w-4 text-muted-foreground" />
      </div>
    )}
    {item.campaignName}
    {item.creative?.objectType && (
      <Badge variant="outline" className="text-[9px]">
        {item.creative.objectType === 'VIDEO' ? 'Video' : item.creative.objectType === 'CAROUSEL' ? 'Carousel' : 'Imagem'}
      </Badge>
    )}
  </span>
</td>
```

### Lightbox Component

```typescript
// Usar shadcn Dialog
<Dialog open={lightboxOpen} onOpenChange={setLightboxOpen}>
  <DialogContent className="max-w-lg">
    <img src={creative.imageUrl || creative.thumbnailUrl} className="w-full rounded" />
    {creative.title && <h3>{creative.title}</h3>}
    {creative.body && <p>{creative.body}</p>}
    {creative.objectType === 'VIDEO' && (
      <a href={`https://www.facebook.com/ads/library/?id=${adId}`} target="_blank">
        Ver no Meta →
      </a>
    )}
  </DialogContent>
</Dialog>
```
