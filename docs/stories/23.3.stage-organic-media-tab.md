# Story 23.3 — Tab "Mídias Orgânicas" no Stage Page

## Status

Draft

## Story

**As a** membro do projeto,
**I want** uma nova aba "Mídias Orgânicas" dentro do dashboard da etapa do funil que mostra todos os vídeos do YouTube e posts do Instagram vinculados àquela etapa, com métricas atualizadas,
**so that** eu veja todo o esforço de conteúdo orgânico que está sustentando aquela etapa, lado a lado com o que já mostro de Meta Ads, YouTube Ads, Switchy etc.

## Acceptance Criteria

### Tab Trigger

1. Nova `<TabsTrigger value="organic-media">` adicionada em `packages/web/app/(app)/projects/[id]/funnels/[funnelId]/stages/[stageId]/page.tsx`:
   - Posição: depois da aba "Lead Scoring" (criada na 22.1) — última aba à direita
   - Ícone: `Sparkles` ou `Megaphone` (lucide-react) — escolher o que se encaixa melhor visualmente com as outras abas (decidir no dev, registrar em comentário)
   - Label: "Mídias Orgânicas"
   - Badge de contagem: `{count}` quando count > 0 (mesmo padrão das abas Meta/YouTube)

### Componente da Tab

2. Criado `packages/web/components/funnels/organic-media-tab.tsx`:
   - Props: `{ projectId: string, funnelId: string, stageId: string }`
   - Consome `useStageOrganicPosts(projectId, funnelId, stageId)` (hook novo, story 23.3)
   - Renderiza grid de cards (`grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-3`)

### Cards (multi-source)

3. **Card YouTube** (quando `source === 'youtube'`):
   - Thumbnail (16:9) com badge `Youtube` vermelho no canto superior esquerdo
   - Título (truncado 2 linhas)
   - Linha de métricas: `👁 {viewCount}` · `👍 {likeCount}` · `💬 {commentCount}`
   - Linha secundária (se disponível): `⏱ {watchTimeMinutes}min` · `📊 {avgRetention}%`
   - Click no card → abre `youtube.com/watch?v={externalId}` em nova aba
   - Botão `Trash2` ou `X` no canto superior direito → desvincula (com confirm via `<AlertDialog>`)

4. **Card Instagram** (quando `source === 'instagram'`):
   - Thumbnail (1:1 — Instagram default) com badge `Instagram` rosa/roxo
   - Caption truncada (`title` do payload)
   - Linha de métricas: `👁 {reach}` · `❤ {likeCount}` · `💬 {commentCount}` · `🔖 {saved ?? '—'}`
   - Click no card → abre URL do post em nova aba (`instagram.com/p/...` — fonte do `externalUrl` do payload)
   - Botão de desvincular igual ao YouTube

### Estados especiais

5. **Post órfão (`isStale: true`):** card renderiza em opacity 50%, com banner "Post indisponível na fonte" sobre o thumbnail (`thumbnailUrl: null` → placeholder cinza), métricas mostram `—`. Botão de desvincular permanece funcional (única ação útil).

6. **Hidratação falhou (`hydration: null`, mas link existe):** mesmo tratamento do órfão, mas banner diz "Não foi possível carregar dados — tente recarregar". Botão `RefreshCw` no canto força re-fetch da query.

7. **Loading:** skeleton de 3 cards enquanto fetch.

8. **Empty state:** mensagem centralizada "Nenhuma mídia orgânica vinculada a esta etapa. Vá para o Dashboard de Orgânico para vincular vídeos do YouTube ou posts do Instagram." com link `<Link href={`/projects/${projectId}/youtube-organic`}>` para o dash de YouTube e (se Instagram tiver dash dedicada do projeto) link análogo.

### Hooks Web

9. Novo hook `useStageOrganicPosts(projectId, funnelId, stageId)` em `packages/web/lib/hooks/use-organic-posts.ts` (mesmo arquivo da story 23.2):
   - Query key: `['stage-organic-posts', projectId, funnelId, stageId]`
   - GET `/api/projects/:id/funnels/:fid/stages/:sid/organic-posts`
   - Refetch on window focus: false (cache de 5min do servidor já basta)

10. `useUnlinkOrganicPost` (já criado na 23.2) é reusado aqui para o botão de desvincular dos cards. Após sucesso, invalida `['stage-organic-posts', ...]` E `['organic-post-links', projectId, source]` (para o badge no Dash Orgânico atualizar).

### Filtro & Ordem

11. Posts ordenados por `createdAt` desc (mais recente primeiro). Sem filtro/sort manual na v1.

12. Toggle de filtro por source no topo da tab (3 botões: "Todos" / "YouTube" / "Instagram") — UI só, filtragem client-side.

### Validação

13. `pnpm --filter @loyola-x/web typecheck` → 0 erros
14. `pnpm --filter @loyola-x/web lint` → 0 erros
15. Smoke test:
    - Vincular 1 vídeo YouTube + 1 post Instagram a uma etapa (via UI da 23.2)
    - Abrir Stage Page → aba "Mídias Orgânicas" → ambos aparecem com métricas
    - Click no card → abre fonte externa em nova aba
    - Desvincular → confirma → card some
    - Toggle "YouTube" → só YouTube renderiza
    - Toggle "Todos" → ambos voltam

## Dev Notes

### Componentes existentes a estudar

- `<LeadScoringTab>` em `packages/web/components/funnels/lead-scoring-tab.tsx` (story 22.1) — padrão mais recente de tab dentro do stage page
- `<SwitchyLinksTab>` — padrão de listagem de itens vinculados à etapa
- Cards do `<YouTubeFunnelSection>` — possível inspiração visual para os cards de YouTube

### Riscos UX

- **Mistura de aspect ratios** (YouTube 16:9 vs Instagram 1:1) no grid pode ficar visualmente irregular. Mitigação: forçar todos os cards a `aspect-video` no thumbnail (Instagram terá object-cover) — DEC: dev pode mudar pra `aspect-square` se ficar feio com vídeos do YouTube. Registrar decisão.
- **Performance:** se uma etapa tiver 50+ posts vinculados, hidratação fica lenta. Mitigação: paginação client-side (mostrar 20 primeiros, "Carregar mais") — fica para v2 se virar problema.

### Dependência

- **Story 23.1 Done:** endpoints + shared types
- **Story 23.2 Done:** hooks `useUnlinkOrganicPost`, `use-organic-posts.ts` arquivo já criado
