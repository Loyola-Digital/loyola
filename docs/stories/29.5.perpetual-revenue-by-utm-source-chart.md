# Story 29.5 — Gráfico "Receita por Origem (UTM Source)"

**Epic:** 29
**Status:** Ready for Review
**Points:** 2
**Depende de:** 29.3

## User Story

Como usuário de funil perpétuo com planilha conectada, quero ver um gráfico de barras horizontais mostrando **Receita por Origem (utm_source)** ao lado dos gráficos existentes de Canal/Público/Criativo, para identificar visualmente quais fontes específicas (google, instagram, parcerias, etc.) trazem mais receita.

- [x] **AC2:** Quando planilha NÃO conectada, comportamento atual preservado (3 colunas).

- [x] **AC5:** Typecheck passa.

## Tasks

- [ ] 1. Em `perpetual-dashboard.tsx`, criar `revenueByUtmSource` `useMemo`:
  ```ts
  const revenueByUtmSource = useMemo(() => {
    if (!usingSpreadsheet || !salesData) return [];
    return salesData.porUtmSource
      .slice(0, 8)
      .map((u) => ({
        name: u.source.length > 25 ? u.source.slice(0, 25) + "..." : u.source,
        revenue: u.bruto,
      }));
  }, [salesData, usingSpreadsheet]);
  ```
- [ ] 2. Trocar o grid de 3 colunas pra 4 quando `usingSpreadsheet`:
  ```tsx
  <div className={`grid grid-cols-1 ${usingSpreadsheet ? "lg:grid-cols-4" : "lg:grid-cols-3"} gap-6`}>
    {usingSpreadsheet && (
      <HBarChart
        title="Receita por Origem"
        data={revenueByUtmSource}
        funnelContext={...}
        entityType="campaign" // sem tooltip de Meta — origem é meta-agnóstica
      />
    )}
    <HBarChart title="Receita por Canal" ... />
    <HBarChart title="Receita por Público" ... />
    <HBarChart title="Receita por Criativo" ... />
  </div>
  ```
- [ ] 3. Typecheck.
