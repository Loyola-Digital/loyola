## Story

**As a** Loyola Digital admin/manager,
**I want** a settings interface to connect a Google Sheets spreadsheet to a project, preview tab data, and map columns (UTM Campaign, UTM Medium, UTM Content, etc.) for each tab type (leads, survey, sales),
**so that** the platform knows which spreadsheet data to pull and how to match it to Meta Ads campaigns.

---

### Column Mapping UI

Para cada aba mapeada, mostrar:
1. Preview table (primeiras 10 linhas) — ajuda o usuario a ver quais colunas existem
2. Dropdown selectors por campo obrigatorio do tipo:
   - **Leads:** UTM Campaign (select coluna), UTM Medium (select coluna), UTM Content (select coluna)
   - **Vendas:** UTM Campaign, UTM Medium, UTM Content, Valor da venda
   - **Pesquisa:** Campos dinamicos — o usuario clica "Adicionar campo" e seleciona coluna + label
3. Colunas disponiveis no dropdown vem do header row (row 0 do preview)

### Hook Pattern

```typescript
// Adicionar em novo arquivo: packages/web/lib/hooks/use-google-sheets.ts

export function useGoogleSheetsConnection(projectId: string) {
  const apiClient = useApiClient();
  return useQuery({
    queryKey: ["google-sheets-connection", projectId],
    queryFn: () => apiClient(`/api/google-sheets/connections/${projectId}`),
    enabled: !!projectId,
  });
}

export function useConnectGoogleSheet() {
  const apiClient = useApiClient();
  const queryClient = useQueryClient();
  return useMutation({
    mutationFn: (data: { projectId: string; spreadsheetUrl: string }) =>
      apiClient("/api/google-sheets/connections", { method: "POST", body: data }),
    onSuccess: (_, vars) =>
      queryClient.invalidateQueries({ queryKey: ["google-sheets-connection", vars.projectId] }),
  });
}

export function useSheetTabPreview(connectionId: string, tabName: string) {
  const apiClient = useApiClient();
  return useQuery({
    queryKey: ["sheet-tab-preview", connectionId, tabName],
    queryFn: () => apiClient(`/api/google-sheets/connections/${connectionId}/tabs/${encodeURIComponent(tabName)}/preview`),
    enabled: !!connectionId && !!tabName,
  });
}
```
