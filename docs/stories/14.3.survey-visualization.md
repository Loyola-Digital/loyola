# Story 14.3 — Visualizacao de Pesquisas no Funil

**Epic:** EPIC-14 Google Sheets — Pesquisas de Funil
**Status:** Draft
**Priority:** Medium
**Estimate:** L
**Depends on:** 14.2

---

## Story

Como usuario do Loyola X, quero ver os dados das pesquisas vinculadas dentro do funil, com a tabela de respostas e resumo, para analisar feedback dos leads.

## Acceptance Criteria

### AC1: Secao colapsavel por pesquisa
- [ ] Cada pesquisa (aba) aparece como secao colapsavel
- [ ] Header mostra: nome da planilha + nome da aba + total de respostas
- [ ] Clique expande para ver dados

### AC2: Tabela de respostas
- [ ] Headers da planilha como colunas da tabela
- [ ] Rows renderizadas como linhas
- [ ] Paginacao (20 por pagina) se tiver muitas respostas
- [ ] Scroll horizontal se tiver muitas colunas

### AC3: Resumo por coluna
- [ ] Para colunas com respostas repetidas (ex: "Sim/Nao", escalas 1-5), mostrar contagem/porcentagem
- [ ] Mini bar chart horizontal para distribuicao de respostas
- [ ] Detectar automaticamente colunas categoricas vs texto livre

### AC4: Refresh
- [ ] Botao para recarregar dados da planilha (limpar cache)
- [ ] Indicador de quando foi ultima leitura

### AC5: Multiplas pesquisas
- [ ] Se o funil tem N pesquisas, cada uma aparece como secao separada
- [ ] Ordenadas por nome da planilha

## Tasks

- [ ] 1. Criar componente `SurveySection` (colapsavel com header + dados)
- [ ] 2. Criar componente de tabela de respostas com paginacao
- [ ] 3. Criar componente de resumo por coluna (bar chart distribuicao)
- [ ] 4. Detector de tipo de coluna (categorica vs texto)
- [ ] 5. Integrar na tab Pesquisas do funil

## Dev Notes

- Dados vem do endpoint `GET /api/google-sheets/spreadsheets/:id/sheets/:name/data`
- Header = primeira linha, rows = demais
- Para resumo categorico: agrupar valores unicos e contar ocorrencias
- Threshold para categorico: se coluna tem <= 10 valores unicos, tratar como categorica
- Cache de 5min no backend
