# Story 29.2 — UI Wizard de Conexão de Planilha no PerpetualDashboard

**Epic:** 29 — Perpetual Spreadsheet Integration
**Status:** Ready for Review
**Points:** 3
**Depende de:** 29.1

## User Story

Como **usuário com funil de tipo perpétuo**, quero clicar em um botão no header do dashboard, conectar minha planilha Google Sheets de vendas com mapeamento de colunas (incluindo utm_source), e ter um indicador visual de que a planilha está conectada — para futuras stories usarem esses dados nos KPIs e gráficos do dashboard.

## Dev Notes

### Onde encaixar o botão

O header atual do `PerpetualDashboard` está em `perpetual-dashboard.tsx:227-236`:

```tsx
<div className="flex items-center justify-between gap-3 flex-wrap">
  <div className="flex items-center gap-2 flex-wrap">
    <DayRangePicker days={days} onDaysChange={setDays} />
    <RefreshDataButton />
  </div>
  <Button ... onClick={() => setShowCampaignManager(!showCampaignManager)}>
    {funnel.campaigns.length} campanhas
  </Button>
</div>
```

Adicionar o botão de planilha logo após `<RefreshDataButton />` no mesmo flex container da esquerda.

### Pattern de invalidação

Quando usuário troca a planilha conectada, o hook `useConnectPerpetualSpreadsheet` invalida tanto a query de metadata quanto a (futura) query de dados agregados, igual o `invalidateStageSalesData` faz.

### Component reuse

`PerpetualSpreadsheetWizardDialog` é 90% clone do `StageSalesWizardDialog`. Não vale extrair em base component agora (YAGNI) — quando uma terceira variante aparecer, refatoramos.
